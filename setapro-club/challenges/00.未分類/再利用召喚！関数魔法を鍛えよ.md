## 「再利用召喚！関数魔法を鍛えよ」

## ミッション：3つの魔法ボタンから、同じ仕組みで違うエフェクトを出そう！

この課題では「同じ処理を関数にまとめる」「引数で内容だけ差し替えて使い回す」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">使う魔法を選ぼう</p>
  <div class="buttons">
    <button id="fireBtn">🔥 炎</button>
    <button id="iceBtn">🧊 氷</button>
    <button id="healBtn">✨ 回復</button>
  </div>
  <p id="log">ボタンを押すとエフェクトが出ます</p>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const fireBtn = document.querySelector('#fireBtn');
const iceBtn = document.querySelector('#iceBtn');
const healBtn = document.querySelector('#healBtn');
const log = document.querySelector('#log');

function castBurst(x, y, emoji, color) {
  for (let i = 0; i < 6; i++) {
    const particle = document.createElement('div');
    particle.textContent = emoji;
    particle.classList.add('particle');
    particle.style.left = x + 'px';
    particle.style.top  = y + 'px';
    particle.style.color = color;

    const dx = (Math.random() - 0.5) * 120;
    const dy = -(Math.random() * 90 + 40);
    particle.style.setProperty('--dx', dx + 'px');
    particle.style.setProperty('--dy', dy + 'px');

    document.body.appendChild(particle);
    particle.classList.add('flying');

    setTimeout(function() {
      particle.remove();
    }, 800);
  }
}

fireBtn.addEventListener('click', function(event) {
  castBurst(event.clientX, event.clientY, '🔥', '#ff7a00');
  log.textContent = '炎の魔法を発動しました';
});

iceBtn.addEventListener('click', function(event) {
  castBurst(event.clientX, event.clientY, '🧊', '#57c7ff');
  log.textContent = '氷の魔法を発動しました';
});

healBtn.addEventListener('click', function(event) {
  castBurst(event.clientX, event.clientY, '✨', '#ffe066');
  log.textContent = '回復魔法を発動しました';
});
```

3つのボタンが、それぞれ違う絵文字と色でエフェクトを出すことを確認してください。

### コードを読み解こう

#### 同じ処理を3回コピペしない

今回の3つのボタンは、違うのは「絵文字」と「色」だけです。要素を作る処理、飛ばす処理、消す処理は全部同じです。

そこで、共通部分を1つの関数にまとめます。

```js
function castBurst(x, y, emoji, color) {
  // 共通の処理
}
```

こうすると、同じようなコードを何回も書かずに済みます。

#### 関数の引数：あとから差し込む値の置き場所

```js
function castBurst(x, y, emoji, color) {
```

`x`、`y`、`emoji`、`color` は**引数**です。あとから関数を呼ぶときに、ここへ値を渡します。

```js
castBurst(event.clientX, event.clientY, '🔥', '#ff7a00');
```

この呼び出しでは、次のように対応しています。

| 渡した値 | 関数の中で入る場所 |
|----------|--------------------|
| `event.clientX` | `x` |
| `event.clientY` | `y` |
| `'🔥'` | `emoji` |
| `'#ff7a00'` | `color` |

#### 関数の中では、受け取った値をそのまま使える

```js
particle.textContent = emoji;
particle.style.color = color;
particle.style.left = x + 'px';
particle.style.top  = y + 'px';
```

これで、同じ処理なのに「炎なら炎の見た目」「氷なら氷の見た目」に変えられます。

#### 呼び出す側は短く書ける

```js
iceBtn.addEventListener('click', function(event) {
  castBurst(event.clientX, event.clientY, '🧊', '#57c7ff');
});
```

ボタンごとの差は、`castBurst(...)` の中の値だけです。これが関数化の強みです。

### 改造しよう

#### ミッション①：4つ目の魔法を追加しよう

たとえば「雷」のボタンを追加してみましょう。

```js
castBurst(event.clientX, event.clientY, '⚡', '#ffd43b');
```

#### ミッション②：出る数を変えよう

```js
for (let i = 0; i < 6; i++) {
```

この `6` を変えると、1回で出る個数が変わります。

#### ミッション③：ログの文言を変えよう

```js
log.textContent = '炎の魔法を発動しました';
```

「超回復！」「氷結中！」のように、雰囲気のある文に変えてみましょう。

✅️ ボタンごとの差分だけをイベントの中に書いている
