## 「小テスト：再利用召喚！関数魔法を鍛えよ」

## ミッション：同じ処理を関数にまとめて、短く書けるか確認しよう！

この小テストでは、本編で学んだ「関数」「引数」「呼び出し」を使います。

### 解くときの約束

- コードは全部書き直さず、必要なところだけ直してください
- 同じ処理を2回以上書く前に、関数にできないか考えてください


### 問1：ウォームアップ

次の関数を完成させてください。

```html
<p id="log">ここに結果が出ます</p>
```

```js
const log = document.querySelector('#log');

function showMessage(text) {
  // TODO
}

showMessage('炎の魔法');
```

#### 条件

- 変更は `1行`
- `#log` に文字を表示する

#### 確認ポイント

✅️ `炎の魔法` と表示される
✅️ 関数の引数 `text` を使えている


### 問2：ミニ実装

次のコードは、同じような処理を2回書いています。

```html
<button id="fireBtn">炎</button>
<button id="iceBtn">氷</button>
<p id="log">まだ発動していません</p>
```

```js
const fireBtn = document.querySelector('#fireBtn');
const iceBtn = document.querySelector('#iceBtn');
const log = document.querySelector('#log');

function castSpell(name) {
  log.textContent = name + ' を発動しました';
}

fireBtn.addEventListener('click', function() {
  // TODO
});

iceBtn.addEventListener('click', function() {
  // TODO
});
```

#### お題

2つのボタンから、同じ `castSpell()` を呼び出してください。

#### 条件

- 変更は `2行`
- 炎は `炎`
- 氷は `氷`

#### 確認ポイント

✅️ 炎ボタンで `炎 を発動しました` と出る
✅️ 氷ボタンで `氷 を発動しました` と出る


### 問3：できたら挑戦

次のコードに、色も変えられる関数を追加してください。

```html
<button id="healBtn">回復</button>
<p id="log">まだ発動していません</p>
```

```js
const healBtn = document.querySelector('#healBtn');
const log = document.querySelector('#log');

function castSpell(name, color) {
  log.textContent = name + ' を発動しました';
}

healBtn.addEventListener('click', function() {
  castSpell('回復', 'green');
});
```

#### お題

`castSpell()` の中で、`log.style.color = color;` も実行してください。

#### ヒント

引数 `color` は関数の中でそのまま使えます。

#### 確認ポイント

✅️ 回復を押すと文字色が変わる
✅️ 関数の引数を2つ使えている
