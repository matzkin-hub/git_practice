## 「時間制御！インターバル魔法を刻め」

## ミッション：10秒間だけエナジーを自動でため続ける装置を作ろう！

この課題では「`setInterval()` で一定間隔の処理をくり返す」「`clearInterval()` で止める」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">エナジーチャージ装置</p>
  <button id="startBtn">▶ チャージ開始</button>
  <p class="value">エナジー：<span id="energyDisplay">0</span></p>
  <p class="value">残り時間：<span id="timeDisplay">10</span>秒</p>
  <p id="message">ボタンを押すと10秒だけ動きます</p>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const startBtn = document.querySelector('#startBtn');
const energyDisplay = document.querySelector('#energyDisplay');
const timeDisplay = document.querySelector('#timeDisplay');
const message = document.querySelector('#message');

let energy = 0;
let timeLeft = 10;
let energyInterval;
let timerInterval;

startBtn.addEventListener('click', function() {
  energy = 0;
  timeLeft = 10;
  energyDisplay.textContent = energy;
  timeDisplay.textContent = timeLeft;
  message.textContent = 'チャージ中です';
  startBtn.disabled = true;

  energyInterval = setInterval(function() {
    energy = energy + 3;
    energyDisplay.textContent = energy;
  }, 500);

  timerInterval = setInterval(function() {
    timeLeft = timeLeft - 1;
    timeDisplay.textContent = timeLeft;

    if (timeLeft === 0) {
      clearInterval(energyInterval);
      clearInterval(timerInterval);
      startBtn.disabled = false;
      message.textContent = 'チャージ完了！ ' + energy + ' エナジーたまりました';
    }
  }, 1000);
});
```

ボタンを押すと、10秒間だけエナジーが自動で増えることを確認してください。

### コードを読み解こう

#### `setInterval()` ：一定時間ごとに同じ処理をくり返す

```js
energyInterval = setInterval(function() {
  energy = energy + 3;
}, 500);
```

このコードは、500ミリ秒ごとにエナジーを3ずつ増やします。

`setTimeout()` は1回だけですが、`setInterval()` は止めるまで何度も動きます。

#### インターバルは変数に入れておく

```js
let energyInterval;
let timerInterval;
```

あとで止めるために、`setInterval()` の戻り値を変数に入れています。

#### `clearInterval()` で止める

```js
clearInterval(energyInterval);
clearInterval(timerInterval);
```

止める処理を書かないと、終わってほしい場面でもずっと動き続けます。

今回は、

- エナジーを増やすインターバル
- 時間を減らすインターバル

の2つを別々に動かして、最後に両方止めています。

#### 2つの時間を同時に扱う

| インターバル | 間隔 | 役割 |
|--------------|------|------|
| `energyInterval` | 500ms | エナジーを増やす |
| `timerInterval` | 1000ms | 残り時間を減らす |

役割ごとに分けると、コードの意味が分かりやすくなります。

### 改造しよう

#### ミッション①：増える量を変えよう

```js
energy = energy + 3;
```

1回ごとの増加量を変えると、チャージ速度が変わります。

#### ミッション②：動く時間を変えよう

```js
let timeLeft = 10;
```

5秒チャレンジや30秒チャレンジにもできます。

#### ミッション③：エナジーが一定値を超えたら色を変えよう

`energy >= 30` のような条件を使うと、盛り上がり演出を作れます。


#### チェックしてみよう

✅️ `setInterval()` を使っている
✅️ エナジーが自動で増える
✅️ 残り時間が自動で減る
✅️ 10秒後に `clearInterval()` で止まる
