## 「操作覚醒！キーボード魔法を解放せよ」

## ミッション：矢印キーでキャラクターを上下左右に動かそう！

この課題では「キーボード入力を受け取る」「座標を更新して要素を動かす」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">矢印キーで勇者を動かしてみよう</p>

<div id="stage">
  <div id="player">🧙</div>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const player = document.querySelector('#player');

let x = 160;
let y = 160;
const step = 20;

function renderPlayer() {
  player.style.left = x + 'px';
  player.style.top = y + 'px';
}

renderPlayer();

document.addEventListener('keydown', function(event) {
  switch (event.key) {
    case 'ArrowUp':
      y = y - step;
      break;
    case 'ArrowDown':
      y = y + step;
      break;
    case 'ArrowLeft':
      x = x - step;
      break;
    case 'ArrowRight':
      x = x + step;
      break;
    default:
      return;
  }

  event.preventDefault();

  if (x < 20) {
    x = 20;
  }
  if (x > 300) {
    x = 300;
  }
  if (y < 20) {
    y = 20;
  }
  if (y > 300) {
    y = 300;
  }

  renderPlayer();
});
```

矢印キーを押すと、勇者がステージの中を動くことを確認してください。

### コードを読み解こう

#### `keydown` ：キーが押された瞬間に反応する

```js
document.addEventListener('keydown', function(event) {
```

`click` の代わりに `keydown` を使うと、キーボードが押されたことを受け取れます。

#### `event.key` ：どのキーが押されたかを調べる

```js
switch (event.key) {
  case 'ArrowUp':
```

矢印キーの場合は、

- `ArrowUp`
- `ArrowDown`
- `ArrowLeft`
- `ArrowRight`

という文字列が入ります。

#### 座標を変えてから描き直す

```js
x = x + step;
y = y - step;
renderPlayer();
```

今回は勇者そのものを直接動かすのではなく、まず `x` と `y` という数字を更新しています。

そのあと `renderPlayer()` で、

```js
player.style.left = x + 'px';
player.style.top = y + 'px';
```

を実行して、画面に反映しています。

#### 画面の外へ出ないようにする

```js
if (x < 20) {
  x = 20;
}
```

このように下限と上限を作ると、ステージの外へ出なくなります。

### 改造しよう

#### ミッション①：移動量を変えよう

```js
const step = 20;
```

大きくすると速く動き、小さくすると細かく動けます。

#### ミッション②：キャラクターを変えよう

```html
<div id="player">🧙</div>
```

好きな絵文字に変えてみましょう。

#### ミッション③：壁を作ろう

特定の場所では止まるようにすると、迷路のようなステージにもできます。


#### チェックしてみよう

✅️ `keydown` でキーボード入力を受け取っている
✅️ `event.key` で押されたキーを判定している
✅️ `x` と `y` の座標で移動を管理している
