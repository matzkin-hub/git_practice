## 「武器庫整列！配列魔法を解放せよ」

## ミッション：宝箱を開くたびに、ランダムなアイテムを獲得して履歴に追加しよう！

この課題では「配列にデータをためる」「`push()` と `length` で状態を管理する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">宝箱を開けよう</p>
  <button id="openBtn">🎁 宝箱を開く</button>
  <p class="latest">最新の入手：<span id="latestItem">まだ何もありません</span></p>
  <p class="count">所持数：<span id="itemCount">0</span>個</p>
  <ul id="inventoryList"></ul>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const openBtn = document.querySelector('#openBtn');
const latestItem = document.querySelector('#latestItem');
const itemCount = document.querySelector('#itemCount');
const inventoryList = document.querySelector('#inventoryList');

const lootTable = [
  '炎のつえ',
  '氷のゆびわ',
  '雷のペンダント',
  '回復の薬',
  'すばやさのくつ',
  '星くずのかけら'
];

const inventory = [];

openBtn.addEventListener('click', function() {
  const randomIndex = Math.floor(Math.random() * lootTable.length);
  const item = lootTable[randomIndex];

  inventory.push(item);

  latestItem.textContent = item;
  itemCount.textContent = inventory.length;

  inventoryList.textContent = '';

  for (let i = 0; i < inventory.length; i++) {
    const li = document.createElement('li');
    li.textContent = inventory[i];
    inventoryList.appendChild(li);
  }
});
```

宝箱を開くたびに、ランダムなアイテムが追加されて一覧が増えることを確認してください。

### コードを読み解こう

#### `lootTable` ：候補を並べた配列

```js
const lootTable = [
  '炎のつえ',
  '氷のゆびわ',
  '雷のペンダント',
  '回復の薬',
  'すばやさのくつ',
  '星くずのかけら'
];
```

配列は、複数のデータを順番に並べて入れておける箱です。

今回の `lootTable` には「どんなアイテムが出るか」の候補を入れています。

#### `inventory` ：実際に手に入れたものをためる配列

```js
const inventory = [];
```

こちらは最初は空です。宝箱を開くたびに、手に入れたアイテムをここへ追加していきます。

#### `push()` ：配列の最後に追加する

```js
inventory.push(item);
```

`push()` は、配列のいちばん後ろに新しいデータを追加するメソッドです。

1回目に `回復の薬` を引いたら：

```js
['回復の薬']
```

2回目に `炎のつえ` を引いたら：

```js
['回復の薬', '炎のつえ']
```

のように増えていきます。

#### `length` ：今いくつ入っているかを数える

```js
itemCount.textContent = inventory.length;
```

`length` は配列の要素数です。宝箱を開くたびに `inventory.length` が増えるので、そのまま所持数の表示に使えます。

#### 配列の中身を `for` ループで一覧表示する

```js
inventoryList.textContent = '';

for (let i = 0; i < inventory.length; i++) {
  const li = document.createElement('li');
  li.textContent = inventory[i];
  inventoryList.appendChild(li);
}
```

いったん一覧を空にしてから、配列の先頭から順番に `<li>` を作り直しています。

`inventory[i]` は「配列の `i` 番目の要素」という意味です。

### 改造しよう

#### ミッション①：宝箱の中身を増やそう

```js
const lootTable = [
  '炎のつえ',
  '氷のゆびわ'
];
```

好きなアイテム名を追加してみましょう。

#### ミッション②：最新のアイテムを目立たせよう

```js
latestItem.textContent = item;
```

この表示の色や大きさを変えると、何が出たか分かりやすくなります。

#### ミッション③：一覧に番号を付けよう

```js
li.textContent = inventory[i];
```

ここを `i + 1` と組み合わせると、「1個目」「2個目」と番号付きで表示できます。

✅️ `inventory.push(...)` を使っている
