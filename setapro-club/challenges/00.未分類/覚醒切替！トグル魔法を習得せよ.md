## 「覚醒切替！トグル魔法を習得せよ」

## ミッション：ボタンを押すたびに、お気に入り状態がONとOFFで切り替わるようにしよう！

この課題では「`true / false` で状態を持つ」「状態に応じて文字と見た目を切り替える」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="card">
  <p class="label">今日の推しショット</p>
  <button id="favBtn">☆ お気に入りに入れる</button>
  <p id="status">いまは未登録です</p>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  min-height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  background: linear-gradient(180deg, #fff7fb, #ffeaf2);
  font-family: sans-serif;
}

.card {
  width: 320px;
  padding: 32px;
  border-radius: 24px;
  background: white;
  text-align: center;
  box-shadow: 0 18px 40px rgba(0, 0, 0, 0.08);
}

.label {
  margin-top: 0;
  margin-bottom: 24px;
  font-size: 1.2rem;
  font-weight: bold;
  color: #333;
}

#favBtn {
  border: none;
  border-radius: 999px;
  padding: 16px 28px;
  font-size: 1.1rem;
  background: #f3f4f6;
  color: #666;
  cursor: pointer;
  transition: all 0.2s;
}

#favBtn.active {
  background: #e85088;
  color: white;
  transform: scale(1.05);
}

#status {
  margin-bottom: 0;
  margin-top: 20px;
  color: #888;
}

#status.active {
  color: #e85088;
  font-weight: bold;
}
```

JSタブに下のコードを貼り付けてください。

```js
const favBtn = document.querySelector('#favBtn');
const status = document.querySelector('#status');

let isFavorite = false;

favBtn.addEventListener('click', function() {
  isFavorite = !isFavorite;

  if (isFavorite) {
    favBtn.textContent = '★ お気に入り済み';
    favBtn.classList.add('active');
    status.textContent = 'お気に入りに登録されています';
    status.classList.add('active');
  } else {
    favBtn.textContent = '☆ お気に入りに入れる';
    favBtn.classList.remove('active');
    status.textContent = 'いまは未登録です';
    status.classList.remove('active');
  }
});
```

ボタンを押すたびに、お気に入り状態が切り替わることを確認してください。

### コードを読み解こう

#### `let isFavorite = false` ：状態を変数で持つ

```js
let isFavorite = false;
```

`isFavorite` は「いまお気に入り中かどうか」を表す変数です。`false` は「違う」、`true` は「そうです」を表す値です。

このように、ON/OFFのような2択の状態を表す値を**真偽値**といいます。

#### `isFavorite = !isFavorite` ：今の状態をひっくり返す

```js
isFavorite = !isFavorite;
```

`!` は「反対にする」記号です。

| 元の値 | `!` を付けた結果 |
|--------|------------------|
| `true` | `false` |
| `false` | `true` |

つまり、この1行で「お気に入り中なら解除」「未登録なら登録」に切り替えています。

#### `if / else` ：状態によって処理を分ける

```js
if (isFavorite) {
  favBtn.textContent = '★ お気に入り済み';
} else {
  favBtn.textContent = '☆ お気に入りに入れる';
}
```

`if (isFavorite)` は「`isFavorite` が `true` なら」という意味です。`else` はそれ以外、つまり `false` のときに実行されます。

今回はボタンの文字、説明文、色をまとめて切り替えています。

#### `classList.add` / `classList.remove` ：見た目も切り替える

```js
favBtn.classList.add('active');
status.classList.remove('active');
```

文字だけでなく見た目も変えるために、CSSの `active` クラスを付けたり外したりしています。

```css
#favBtn.active {
  background: #e85088;
  color: white;
}
```

`#favBtn.active` は「`id="favBtn"` で、さらに `active` クラスが付いているとき」という意味です。

### 改造しよう

#### ミッション①：ONのときの文言を変えよう

```js
favBtn.textContent = '★ お気に入り済み';
status.textContent = 'お気に入りに登録されています';
```

好きなサービス名やキャラクター名に変えてみましょう。

#### ミッション②：色を変えよう

```css
#favBtn.active {
  background: #e85088;
}

#status.active {
  color: #e85088;
}
```

ピンク以外の色に変えて、雰囲気を変えてみましょう。

#### ミッション③：最初からONにしてみよう

```js
let isFavorite = false;
```

ここを `true` にするとどうなるか試してみましょう。最初の見た目も合わせて直す必要があるか考えるのがポイントです。
