## 「型ある召喚！クラス魔法で使い魔を量産せよ」

## ミッション：同じルールで敵を何体も作って、まとめてダメージを与えよう！

この課題では「クラスで設計図を作る」「`new` でインスタンスを量産する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">召喚訓練場</p>

  <div class="buttons">
    <button id="spawnBtn">👾 敵を召喚</button>
    <button id="attackBtn">⚔ 全体攻撃</button>
  </div>

  <p id="message">敵を召喚してみましょう</p>
  <ul id="enemyList"></ul>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const spawnBtn = document.querySelector('#spawnBtn');
const attackBtn = document.querySelector('#attackBtn');
const message = document.querySelector('#message');
const enemyList = document.querySelector('#enemyList');

class Enemy {
  constructor(name, hp) {
    this.name = name;
    this.hp = hp;
  }

  takeDamage(amount) {
    this.hp = this.hp - amount;

    if (this.hp < 0) {
      this.hp = 0;
    }
  }

  getLabel() {
    if (this.hp === 0) {
      return this.name + ' / たおれた';
    }

    return this.name + ' / HP ' + this.hp;
  }
}

const enemies = [];
let enemyCount = 1;

function renderEnemies() {
  enemyList.textContent = '';

  enemies.forEach(function(enemy) {
    const li = document.createElement('li');
    li.textContent = enemy.getLabel();

    if (enemy.hp === 0) {
      li.classList.add('down');
    }

    enemyList.appendChild(li);
  });
}

spawnBtn.addEventListener('click', function() {
  const hp = Math.floor(Math.random() * 21) + 20;
  const enemy = new Enemy('スライム' + enemyCount, hp);

  enemyCount = enemyCount + 1;
  enemies.push(enemy);

  message.textContent = enemy.name + ' を召喚しました';
  renderEnemies();
});

attackBtn.addEventListener('click', function() {
  if (enemies.length === 0) {
    message.textContent = 'まだ敵がいません';
    return;
  }

  enemies.forEach(function(enemy) {
    enemy.takeDamage(10);
  });

  message.textContent = '全員に10ダメージを与えました';
  renderEnemies();
});
```

敵を何体も召喚できて、全体攻撃ですべてのHPが減ることを確認してください。

### コードを読み解こう

#### `class Enemy` ：敵の設計図を作る

```js
class Enemy {
  constructor(name, hp) {
    this.name = name;
    this.hp = hp;
  }
}
```

クラスは「同じ形のものをたくさん作るための設計図」です。

今回は、どの敵も

- 名前を持つ
- HPを持つ
- ダメージを受ける

という共通ルールで作ります。

#### `new Enemy(...)` ：設計図から実体を作る

```js
const enemy = new Enemy('スライム1', 30);
```

`new` を使うと、クラスから実際のオブジェクトを作れます。これを**インスタンス**といいます。

#### メソッド：そのオブジェクト専用の処理

```js
takeDamage(amount) {
  this.hp = this.hp - amount;
}
```

`takeDamage()` は敵が持っている処理です。`this.hp` のように書くと、「その敵自身のHP」を意味します。

#### 同じクラスから何体も作れる

```js
enemies.push(enemy);
```

同じ `Enemy` クラスから作った敵を配列にためているので、何体増えても同じルールで扱えます。

### 改造しよう

#### ミッション①：敵の種類を増やそう

`スライム` 以外に `ゴーレム` や `ドラゴン` を混ぜると楽しくなります。

#### ミッション②：攻撃力を変えよう

```js
enemy.takeDamage(10);
```

ダメージ量を変えると難易度が変わります。

#### ミッション③：回復メソッドを追加しよう

`heal(amount)` のようなメソッドを作ると、クラスの便利さがもっと分かります。


#### チェックしてみよう

✅️ `class` を使っている
✅️ `constructor` で初期値を受け取っている
✅️ `new` で複数の敵を作っている
✅️ メソッドでHPを更新している
