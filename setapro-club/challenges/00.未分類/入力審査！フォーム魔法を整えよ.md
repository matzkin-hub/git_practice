## 「入力審査！フォーム魔法を整えよ」

## ミッション：名前を入力して、ギルド名簿にメンバー登録できるようにしよう！

この課題では「`form` の送信を受け取る」「空欄をチェックして安全に登録する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">ギルド加入申請</p>

  <form id="entryForm">
    <input id="nameInput" type="text" placeholder="名前を入力" />

    <select id="roleSelect">
      <option value="戦士">戦士</option>
      <option value="魔法使い">魔法使い</option>
      <option value="回復役">回復役</option>
    </select>

    <button type="submit">登録する</button>
  </form>

  <p id="error"></p>
  <ul id="memberList"></ul>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const entryForm = document.querySelector('#entryForm');
const nameInput = document.querySelector('#nameInput');
const roleSelect = document.querySelector('#roleSelect');
const error = document.querySelector('#error');
const memberList = document.querySelector('#memberList');

entryForm.addEventListener('submit', function(event) {
  event.preventDefault();

  const name = nameInput.value.trim();
  const role = roleSelect.value;

  if (name === '') {
    error.textContent = '名前を入力してください';
    return;
  }

  error.textContent = '';

  const li = document.createElement('li');
  li.textContent = role + '：' + name;
  memberList.appendChild(li);

  entryForm.reset();
});
```

名前を入力して送信すると、名簿に追加されることを確認してください。空欄ならエラーが出ることも確認してください。

### コードを読み解こう

#### `submit` ：フォームが送信されたときに動く

```js
entryForm.addEventListener('submit', function(event) {
```

ボタンを押したときだけでなく、入力欄で Enter を押したときも同じように受け取れます。

#### `preventDefault()` ：ページ再読み込みを止める

```js
event.preventDefault();
```

フォームは本来、送信するとページ遷移や再読み込みをしようとします。今回は画面の中で処理したいので、標準の動きを止めています。

#### `trim()` ：前後の空白を取り除く

```js
const name = nameInput.value.trim();
```

`trim()` を使うと、空白だけ入力された場合も空欄として判定できます。

#### 入力チェックをしてから登録する

```js
if (name === '') {
  error.textContent = '名前を入力してください';
  return;
}
```

空欄のまま登録しないように、先にチェックしています。

### 改造しよう

#### ミッション①：文字数チェックを追加しよう

`if (name.length < 2)` のようにすると、短すぎる名前を防げます。

#### ミッション②：職業を増やそう

```html
<option value="戦士">戦士</option>
```

弓使いや忍者など、好きな職業を追加してみましょう。

#### ミッション③：登録件数を表示しよう

`memberList.children.length` を使うと、現在の人数が分かります。


#### チェックしてみよう

✅️ `form` の `submit` を使っている
✅️ `preventDefault()` で再読み込みを止めている
✅️ `trim()` で空欄チェックをしている
✅️ 入力した名前が名簿に追加される
