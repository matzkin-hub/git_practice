**ミッション：タイミング・演出・色を細かく調整して、自分だけのエフェクトに仕上げよう！**

### 磨き上げポイント一覧

細かく調整できるポイントをまとめます。気になるものから順番に試してみましょう。


#### ポイント①：アニメーションの時間を変える

```css
.heart.flying {
  animation: heartFly 1s ease-out forwards;
                      ↑ ここ
}
```

`1s` を変えるとアニメーションのスピードが変わります。合わせて `setTimeout` の `1000` も同じ比率で変えること。

| `animation` | `setTimeout` | 効果 |
|-------------|-------------|------|
| `0.6s` | `600` | キビキビした動き |
| `1s` | `1000` | 標準 |
| `1.5s` | `1500` | ゆったりした動き |


#### ポイント②：ハートをバラバラのタイミングで飛ばす

今は `for` ループで全員が同時に飛び出します。少しずつズラすと自然な散らばりに見えます。

`for` ループの中の `setTimeout` を使って遅延を加えます。



```js
for (let i = 0; i < 8; i++) {
  const delay = i * 50;  // i番目のハートを i×50ミリ秒遅らせる

  setTimeout(function() {
    const heart = document.createElement('div');
    heart.textContent = '♥';
    heart.classList.add('heart');
    heart.style.left = event.clientX + 'px';
    heart.style.top  = event.clientY + 'px';

    const dx = (Math.random() - 0.5) * 120;
    const dy = -(Math.random() * 80 + 60);
    heart.style.setProperty('--dx', dx + 'px');
    heart.style.setProperty('--dy', dy + 'px');

    const size = Math.random() * 16 + 16;
    heart.style.fontSize = size + 'px';

    const colors = ['#e05080', '#ff6b9d', '#ff4444', '#ff8c00', '#ff69b4'];
    heart.style.color = colors[Math.floor(Math.random() * colors.length)];

    document.body.appendChild(heart);
    heart.classList.add('flying');

    setTimeout(function() {
      heart.remove();
    }, 1000);
  }, delay);
}
```

`delay = i * 50` にすると、1個目は即座、2個目は50ms後、3個目は100ms後…と順番に出てきます。


#### ポイント③：飛ぶ高さとひろがりを調整する

```js
const dx = (Math.random() - 0.5) * 120;  // 横方向の広がり
const dy = -(Math.random() * 80 + 60);   // 縦方向の高さ
```

| 変数 | 数字を大きくすると | 数字を小さくすると |
|------|-----------------|-----------------|
| `dx` の `120` | 横に大きく広がる | 横の広がりが狭まる |
| `dy` の `80` | 飛ぶ高さの幅が広がる | 高さが均一に近づく |
| `dy` の `60` | 全体的に高く飛ぶ | 低めに飛ぶ |

#### ポイント⑤：ボタンを押したときにページ全体を揺らす（チャレンジ）

学んだ「JSからクラスを付けてアニメーションを発動させる」技を使います。

CSSに追加：

```css
@keyframes shake {
  0%   { transform: translate(0, 0); }
  25%  { transform: translate(-4px, 0); }
  75%  { transform: translate(4px, 0); }
  100% { transform: translate(0, 0); }
}

.shaking {
  animation: shake 0.2s ease-out;
}
```

JSに追加（`for` ループの前）：

```js
const page = document.querySelector('.page');
page.classList.add('shaking');
page.addEventListener('animationend', function() {
  page.classList.remove('shaking');
});
```

`animationend` はアニメーションが終わったときに発火するイベントです。アニメーション終了後にクラスを外すことで、次にボタンを押したときも同じアニメーションが再生されます。
