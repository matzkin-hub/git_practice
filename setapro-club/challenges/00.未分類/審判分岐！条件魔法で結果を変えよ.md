## 「審判分岐！条件魔法で結果を変えよ」

## ミッション：修行ボタンを押すたびに経験値がたまり、段階ごとに称号が変わるようにしよう！

この課題では「`if / else if / else` で条件分岐する」「`%` を使って節目の回数を判定する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
```


JSタブに下のコードを貼り付けてください。

```js
const trainBtn = document.querySelector('#trainBtn');
const expDisplay = document.querySelector('#expDisplay');
const rankDisplay = document.querySelector('#rankDisplay');
const bonusMessage = document.querySelector('#bonusMessage');

let exp = 0;

trainBtn.addEventListener('click', function() {
  exp = exp + 1;
  expDisplay.textContent = exp;

  if (exp >= 15) {
    rankDisplay.textContent = '伝説の魔導士';
    rankDisplay.style.color = '#ff9ff3';
  } else if (exp >= 10) {
    rankDisplay.textContent = '上級魔導士';
    rankDisplay.style.color = '#ffd166';
  } else if (exp >= 5) {
    rankDisplay.textContent = '中級魔導士';
    rankDisplay.style.color = '#7bdff2';
  } else {
    rankDisplay.textContent = '見習い';
    rankDisplay.style.color = '#9ad1ff';
  }

  if (exp % 5 === 0) {
    bonusMessage.textContent = '節目達成！ボーナス発生です';
    bonusMessage.classList.add('bonus');
  } else {
    const remain = 5 - (exp % 5);
    bonusMessage.textContent = 'あと' + remain + '回で節目ボーナスです';
    bonusMessage.classList.remove('bonus');
  }
});
```

経験値が増えるにつれて称号が変わり、5回ごとにボーナスメッセージが出ることを確認してください。

### コードを読み解こう

#### `if / else if / else` ：条件ごとに結果を分ける

```js
if (exp >= 15) {
  rankDisplay.textContent = '伝説の魔導士';
} else if (exp >= 10) {
  rankDisplay.textContent = '上級魔導士';
} else if (exp >= 5) {
  rankDisplay.textContent = '中級魔導士';
} else {
  rankDisplay.textContent = '見習い';
}
```

経験値の数字によって、どの称号を出すかを変えています。

`else if` を上から順に判定するので、**大きい条件から先に書く**のがポイントです。

#### 条件の順番が大事

もし先に `exp >= 5` を書いてしまうと、`exp` が `15` のときもそこで止まってしまい、`伝説の魔導士` まで進みません。

つまり、今回のように段階があるときは次の順で考えます。

1. いちばん厳しい条件
2. その次の条件
3. いちばんゆるい条件

#### `%` ：割り算の余りを使う

```js
if (exp % 5 === 0) {
```

`%` は割り算の**余り**を求める記号です。

| 式 | 結果 |
|----|------|
| `5 % 5` | `0` |
| `6 % 5` | `1` |
| `10 % 5` | `0` |

つまり `exp % 5 === 0` は「5回ごとの節目かどうか」を調べています。

#### 次の節目まで何回かも計算できる

```js
const remain = 5 - (exp % 5);
```

たとえば経験値が `7` のとき、`7 % 5` は `2` です。そこから `5 - 2` をすると、次の節目まで `3` 回だと分かります。

### 改造しよう

#### ミッション①：称号の段階を増やそう

```js
if (exp >= 15) {
```

20回以上で「大賢者」など、さらに上の称号を足してみましょう。

#### ミッション②：節目の回数を変えよう

```js
if (exp % 5 === 0) {
```

`5` を `3` や `10` に変えると、ボーナスの出るタイミングが変わります。

#### ミッション③：称号ごとに色以外も変えよう

```js
rankDisplay.style.color = '#ffd166';
```

文字の大きさや背景色も変えると、昇格した感じがもっと出ます。
