## 「時刻計測！反応速度魔法を測れ」

## ミッション：合図が出たらすぐ押して、反応速度を計測しよう！

この課題では「`Date.now()` で時刻を記録する」「開始時刻と終了時刻の差を求める」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <button id="startBtn">スタート</button>
  <p id="message">スタートを押して待ってください</p>
  <button id="hitBtn" style="display: none;">いま押せ！</button>
  <p id="result">結果はここに出ます</p>
  <p id="best">ベスト：まだありません</p>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
```

JSタブに下のコードを貼り付けてください。

```js
const startBtn = document.querySelector('#startBtn');
const hitBtn = document.querySelector('#hitBtn');
const message = document.querySelector('#message');
const result = document.querySelector('#result');
const best = document.querySelector('#best');

let startTime = 0;
let bestTime = null;

startBtn.addEventListener('click', function() {
  result.textContent = '結果はここに出ます';
  message.textContent = '準備して...';
  hitBtn.style.display = 'none';
  startBtn.disabled = true;

  const delay = Math.random() * 2000 + 1000;

  setTimeout(function() {
    hitBtn.style.display = 'inline-block';
    message.textContent = 'いまだ！';
    startTime = Date.now();
  }, delay);
});

hitBtn.addEventListener('click', function() {
  const elapsed = Date.now() - startTime;

  result.textContent = elapsed + ' ミリ秒でした';
  hitBtn.style.display = 'none';
  message.textContent = 'もう一度挑戦できます';
  startBtn.disabled = false;

  if (bestTime === null || elapsed < bestTime) {
    bestTime = elapsed;
    best.textContent = 'ベスト：' + bestTime + ' ミリ秒';
  }
});
```

ランダムなタイミングで合図が出て、押した速さがミリ秒で表示されることを確認してください。

### コードを読み解こう

#### `Date.now()` ：今この瞬間の時刻を数字で取る

```js
startTime = Date.now();
```

`Date.now()` は、今の時刻をミリ秒単位の数字で返します。

#### 差を取ると経過時間になる

```js
const elapsed = Date.now() - startTime;
```

合図が出た瞬間の時刻を `startTime` に入れておき、押した瞬間の時刻から引き算すると、反応時間が出ます。

#### ベスト記録も比べられる

```js
if (bestTime === null || elapsed < bestTime) {
  bestTime = elapsed;
}
```

初回は `bestTime` がまだないので `null` です。そのあとは、今の記録が前より速いかどうかを比べています。

#### ランダム待機と時刻計測を組み合わせる

```js
const delay = Math.random() * 2000 + 1000;
setTimeout(function() {
  startTime = Date.now();
}, delay);
```

「いつ合図が出るか分からない」状況を作ることで、本物の反応速度ゲームらしくなります。

### 改造しよう

#### ミッション①：待ち時間の範囲を変えよう

```js
const delay = Math.random() * 2000 + 1000;
```

最短時間と最長時間を変えると、難しさが変わります。

#### ミッション②：100ミリ秒未満なら特別メッセージを出そう

`if (elapsed < 100)` を使うと、超高速判定が作れます。

#### ミッション③：ベスト以外にワーストも記録しよう

最も遅かった記録も残すと、比較がもっと楽しくなります。

✅️ 押した速さがミリ秒で表示される
✅️ `Date.now()` を使っている
✅️ ベスト記録が更新される
