## 「記憶封印！保存魔法を刻み込め」

## ミッション：ハイスコアを保存して、ページを開き直しても記録が残るようにしよう！

この課題では「`localStorage` にデータを保存する」「`JSON.stringify()` と `JSON.parse()` を使って読み書きする」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">連打修行</p>
  <button id="tapBtn">⚡ 連打する</button>
  <button id="saveBtn">💾 記録を保存</button>
  <button id="resetBtn">↺ 今回の記録をリセット</button>

  <p class="value">今回の記録：<span id="scoreDisplay">0</span></p>
  <p class="value">保存された最高記録：<span id="bestDisplay">0</span></p>
  <p id="message">保存すると次回も残ります</p>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const tapBtn = document.querySelector('#tapBtn');
const saveBtn = document.querySelector('#saveBtn');
const resetBtn = document.querySelector('#resetBtn');
const scoreDisplay = document.querySelector('#scoreDisplay');
const bestDisplay = document.querySelector('#bestDisplay');
const message = document.querySelector('#message');

let score = 0;

let saveData = {
  bestScore: 0
};

const savedText = localStorage.getItem('tapSaveData');

if (savedText !== null) {
  saveData = JSON.parse(savedText);
}

bestDisplay.textContent = saveData.bestScore;

tapBtn.addEventListener('click', function() {
  score = score + 1;
  scoreDisplay.textContent = score;
});

saveBtn.addEventListener('click', function() {
  if (score > saveData.bestScore) {
    saveData.bestScore = score;
    localStorage.setItem('tapSaveData', JSON.stringify(saveData));
    bestDisplay.textContent = saveData.bestScore;
    message.textContent = '新記録を保存しました';
  } else {
    message.textContent = '最高記録は更新されませんでした';
  }
});

resetBtn.addEventListener('click', function() {
  score = 0;
  scoreDisplay.textContent = score;
  message.textContent = '今回の記録をリセットしました';
});
```

記録を保存したあと、ページを開き直しても最高記録が残ることを確認してください。

### コードを読み解こう

#### `localStorage` ：ブラウザにデータを残す

```js
localStorage.setItem('tapSaveData', JSON.stringify(saveData));
```

`localStorage` は、ブラウザの中にデータを保存できる場所です。

ページを閉じても、保存した内容はすぐには消えません。

#### オブジェクトはそのままでは保存できない

```js
let saveData = {
  bestScore: 0
};
```

`localStorage` に保存できるのは文字列です。そのため、オブジェクトを保存するときは `JSON.stringify()` で文字列に変えます。

```js
JSON.stringify(saveData)
```

#### 読み戻すときは `JSON.parse()`

```js
const savedText = localStorage.getItem('tapSaveData');

if (savedText !== null) {
  saveData = JSON.parse(savedText);
}
```

`getItem()` で取り出すと文字列で返ってくるので、`JSON.parse()` で元のオブジェクトに戻します。

#### 保存データがないときも考える

```js
if (savedText !== null) {
```

最初の1回目はまだ何も保存されていません。そのとき `getItem()` は `null` を返します。

この確認を入れておくと、初回アクセスでも安全に動きます。

### 改造しよう

#### ミッション①：プレイヤー名も保存しよう

`bestScore` だけでなく、名前も一緒に保存すると記録帳らしくなります。

#### ミッション②：保存データを消すボタンを作ろう

`localStorage.removeItem('tapSaveData')` を使うと、保存を消せます。

#### ミッション③：更新日時も保存しよう

`Date.now()` を使うと、いつ保存したかも残せます。


#### チェックしてみよう

✅️ `localStorage.setItem()` を使っている
✅️ `localStorage.getItem()` で読み戻している
✅️ `JSON.stringify()` と `JSON.parse()` を使っている
✅️ ページを開き直しても最高記録が残る
