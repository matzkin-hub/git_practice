## 「順位整列！ソート魔法で名簿を磨け」

## ミッション：プレイヤー名とスコアを、高い順に並べたランキングを表示しよう！

この課題では「配列の中にオブジェクトを入れる」「`sort()` と `map()` で表示用に整える」技術を習得します。

JSタブに下のコードを貼り付けてください。

```js
const addBtn = document.querySelector('#addBtn');
const rankingList = document.querySelector('#rankingList');

const names = ['ソラ', 'リン', 'ハル', 'ミナ', 'レオ', 'ユナ'];

const records = [
  { name: 'アオ', score: 120 },
  { name: 'ナギ', score: 180 },
  { name: 'ユイ', score: 150 }
];

function renderRanking() {
  records.sort(function(a, b) {
    return b.score - a.score;
  });

  const html = records.map(function(record, index) {
    return '<li><span>' +
      (index + 1) +
      '位 ' +
      record.name +
      '</span><strong>' +
      record.score +
      '点</strong></li>';
  }).join('');

  rankingList.innerHTML = html;
}

renderRanking();

addBtn.addEventListener('click', function() {
  const randomName = names[Math.floor(Math.random() * names.length)];
  const randomScore = Math.floor(Math.random() * 151) + 50;

  records.push({
    name: randomName,
    score: randomScore
  });

  renderRanking();
});
```

ボタンを押すたびに新しい記録が追加され、高得点順に並び替わることを確認してください。

### コードを読み解こう

#### 配列の中にオブジェクトを入れる

```js
const records = [
  { name: 'アオ', score: 120 },
  { name: 'ナギ', score: 180 },
  { name: 'ユイ', score: 150 }
];
```

1人分の情報はオブジェクトです。それを何人分も並べたのがこの配列です。

#### `sort()` ：並び順を決める

```js
records.sort(function(a, b) {
  return b.score - a.score;
});
```

`b.score - a.score` と書くと、高い点数が前に来ます。

たとえば、

- `a.score = 120`
- `b.score = 180`

なら結果は `60` です。正の数なので、`b` が前に来ます。

#### `map()` ：表示用の文字列に変える

```js
const html = records.map(function(record, index) {
  return '<li>...</li>';
}).join('');
```

`map()` は、配列の中身を別の形に変えるメソッドです。

今回は「ランキングのデータ」を「HTMLの文字列」に変えています。

#### `join('')` ：1本の文字列にまとめる

`map()` の結果は配列です。そのままだと `innerHTML` に入れにくいので、`join('')` でつなげて1本のHTML文字列にしています。

### 改造しよう

#### ミッション①：名前の候補を増やそう

```js
const names = ['ソラ', 'リン', 'ハル'];
```

候補を増やすと、ランキングがにぎやかになります。

#### ミッション②：1位だけ特別な色にしよう

`index === 0` を使うと、トップだけ見た目を変えられます。

#### ミッション③：低い順にも並べてみよう

```js
return b.score - a.score;
```

これを逆にすると、低い順に並びます。
