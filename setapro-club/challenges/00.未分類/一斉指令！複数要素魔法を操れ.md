## 「一斉指令！複数要素魔法を操れ」

## ミッション：並んだ敵をまとめて凍らせたり、元に戻したりしよう！

この課題では「同じクラスを持つ複数要素をまとめて取る」「`forEach()` で全員に同じ命令を出す」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="panel">
  <p class="title">敵の群れ</p>

  <div class="buttons">
    <button id="freezeBtn">🧊 全員を凍らせる</button>
    <button id="resetBtn">☀ 元に戻す</button>
  </div>

  <div class="enemy-grid">
    <div class="enemy">👾</div>
    <div class="enemy">👾</div>
    <div class="enemy">👾</div>
    <div class="enemy">👾</div>
    <div class="enemy">👾</div>
    <div class="enemy">👾</div>
  </div>
</div>
```

JSタブに下のコードを貼り付けてください。

```js
const freezeBtn = document.querySelector('#freezeBtn');
const resetBtn = document.querySelector('#resetBtn');
const enemies = document.querySelectorAll('.enemy');

freezeBtn.addEventListener('click', function() {
  enemies.forEach(function(enemy) {
    enemy.classList.add('frozen');
  });
});

resetBtn.addEventListener('click', function() {
  enemies.forEach(function(enemy) {
    enemy.classList.remove('frozen');
  });
});
```

ボタンを押すと、敵全員の見た目がまとめて変わることを確認してください。

### コードを読み解こう

#### `querySelectorAll()` ：同じ条件に合うものを全部取る

```js
const enemies = document.querySelectorAll('.enemy');
```

`querySelector()` は最初の1個だけを取りますが、`querySelectorAll()` は条件に合うものを全部まとめて取ります。

今回の `enemies` は、6体ぶんの敵が入ったリストです。

#### `forEach()` ：1個ずつ順番に処理する

```js
enemies.forEach(function(enemy) {
  enemy.classList.add('frozen');
});
```

`forEach()` は、リストの中身を先頭から1個ずつ取り出して処理します。

今回は、

1. 1体目に `frozen` を付ける
2. 2体目に `frozen` を付ける
3. 3体目に `frozen` を付ける

という流れを、全員分まとめて自動でやっています。

#### 1体だけでなく全員に命令できる

```js
enemy.classList.add('frozen');
enemy.classList.remove('frozen');
```

1体に対してやることは前と同じです。ただし今回は、それを `forEach()` で6回くり返しているのがポイントです。

### 改造しよう

#### ミッション①：敵の数を増やそう

```html
<div class="enemy">👾</div>
```

同じHTMLを増やすだけで、JavaScriptはそのまま使えます。

#### ミッション②：凍結以外の状態も作ろう

たとえば `burning` クラスを作ると、全員が炎上する演出も作れます。

#### ミッション③：1体ずつ違う絵文字にしよう

```html
<div class="enemy">👾</div>
```

敵の見た目を変えても、`.enemy` クラスが付いていれば同じようにまとめて操作できます。
