## ミッション：HP・MP・レベルを1つのデータとしてまとめて管理しよう！

この課題では「関連する情報をオブジェクトにまとめる」「プロパティを書き換えて画面を更新する」技術を習得します。

```html
  <div class="status-row">

    <span id="mpDisplay">30</span>

  </div>

  <p id="message">行動を選んでください</p>
```


JSタブに下のコードを貼り付けてください。

```js
const hpDisplay = document.querySelector('#hpDisplay');
const mpDisplay = document.querySelector('#mpDisplay');
const levelDisplay = document.querySelector('#levelDisplay');
const damageBtn = document.querySelector('#damageBtn');
const healBtn = document.querySelector('#healBtn');
const levelBtn = document.querySelector('#levelBtn');
const message = document.querySelector('#message');

const player = {
  hp: 100,
  mp: 30,
  level: 1
};

function renderStatus() {
  hpDisplay.textContent = player.hp;
  mpDisplay.textContent = player.mp;
  levelDisplay.textContent = player.level;
}

damageBtn.addEventListener('click', function() {
  player.hp = player.hp - 12;

  if (player.hp < 0) {
    player.hp = 0;
  }

  message.textContent = 'ダメージを受けました';
  renderStatus();
});

healBtn.addEventListener('click', function() {
  if (player.mp < 5) {
    message.textContent = 'MPが足りません';
    return;
  }

  player.hp = player.hp + 20;
  player.mp = player.mp - 5;

  if (player.hp > 100) {
    player.hp = 100;
  }

  message.textContent = '回復魔法を使いました';
  renderStatus();
});

levelBtn.addEventListener('click', function() {
  player.level = player.level + 1;
  player.mp = player.mp + 10;
  message.textContent = 'レベルが上がりました';
  renderStatus();
});
```

ボタンを押すと、HP・MP・レベルが正しく変化することを確認してください。

### コードを読み解こう

#### オブジェクト：ひとかたまりの情報を1つにまとめる

```js
const player = {
  hp: 100,
  mp: 30,
  level: 1
};
```

`player` は1人分のステータスです。`hp`、`mp`、`level` のように、関連する情報をまとめて持てます。

もし変数をバラバラにすると、

```js
let hp = 100;
let mp = 30;
let level = 1;
```

のようになります。数が増えるほど管理しにくくなるので、まとまりのある情報はオブジェクトで持つと見通しがよくなります。

#### `player.hp` のようにプロパティを読む

```js
hpDisplay.textContent = player.hp;
```

`player.hp` は「`player` の中の `hp` を読む」という意味です。

同じように、

- `player.mp`
- `player.level`

でそれぞれの値を取り出せます。

#### プロパティは書き換えられる

```js
player.hp = player.hp - 12;
player.level = player.level + 1;
```

左側が「どこを書き換えるか」、右側が「新しい値」です。

#### 画面更新を `renderStatus()` にまとめる

```js
function renderStatus() {
  hpDisplay.textContent = player.hp;
  mpDisplay.textContent = player.mp;
  levelDisplay.textContent = player.level;
}
```

値を書き換えたあとに毎回この関数を呼ぶことで、画面表示とデータの中身をそろえています。

### 改造しよう

#### ミッション①：スタミナや攻撃力を追加しよう

```js
const player = {
  hp: 100,
  mp: 30,
  level: 1
};
```

ここに新しい項目を足して、画面にも表示してみましょう。

#### ミッション②：回復量を変えよう

```js
player.hp = player.hp + 20;
player.mp = player.mp - 5;
```

数字を変えるとバランスが変わります。

#### ミッション③：HPが0のときのメッセージを変えよう

```js
if (player.hp < 0) {
  player.hp = 0;
}
```

HPが0になったら「たおれた！」と表示するようにしてみましょう。
