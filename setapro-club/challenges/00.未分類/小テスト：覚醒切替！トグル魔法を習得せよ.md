### 問2：ミニ実装

次のコードに、ONのときだけ色が変わる機能を追加してください。

```html
<button id="btn">切り替え</button>
<p id="text">OFF</p>
```

```css
.active {
  color: tomato;
  font-weight: bold;
}
```

```js
const btn = document.querySelector('#btn');
const text = document.querySelector('#text');

let isOn = false;

btn.addEventListener('click', function() {
  isOn = !isOn;

  if (isOn) {
    text.textContent = 'ON';
  } else {
    text.textContent = 'OFF';
  }
});
```

#### お題

ONのときだけ `text` に `active` クラスを付けて、OFFのときは外してください。

#### 条件

- 追加は `2行`
- `classList.add` と `classList.remove` を使う

#### 確認ポイント

✅️ ONのときだけ赤くなる
✅️ OFFに戻すと元の見た目に戻る


### 問3：できたら挑戦

次のコードに、3回押したら特別メッセージを出す機能を追加してください。

```js
const btn = document.querySelector('#btn');
const text = document.querySelector('#text');

let isOn = false;
let count = 0;

btn.addEventListener('click', function() {
  isOn = !isOn;
  count = count + 1;

  if (isOn) {
    text.textContent = 'ON';
  } else {
    text.textContent = 'OFF';
  }
});
```

#### お題

3回目に押したときだけ、`text.textContent` を `3回目！` にしてください。

#### ヒント

`if (count === 3)` を使います。

#### 確認ポイント

✅️ 1回目と2回目は普通に切り替わる
✅️ 3回目だけ特別な表示になる
