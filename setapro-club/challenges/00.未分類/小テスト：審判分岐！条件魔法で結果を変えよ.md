### 問2：ミニ実装

次のコードに、5回ごとの節目メッセージを追加してください。

```html
<button id="btn">修行する</button>
<p id="text">まだ修行していません</p>
```

```js
const btn = document.querySelector('#btn');
const text = document.querySelector('#text');

let count = 0;

btn.addEventListener('click', function() {
  count = count + 1;
  text.textContent = count + '回目';
});
```

#### お題

5回目のときだけ、`text.textContent` を `節目達成！` にしてください。

#### 条件

- `%` を使う
- 追加は `2〜3行`

#### 確認ポイント

✅️ 1〜4回目は普通に回数が出る
✅️ 5回目だけ特別な表示になる


### 問3：できたら挑戦

次のコードには、条件の順番に問題があります。

```js
const rank = document.querySelector('#rank');
let exp = 12;

if (exp >= 5) {
  rank.textContent = '中級';
} else if (exp >= 10) {
  rank.textContent = '上級';
} else {
  rank.textContent = '見習い';
}
```

#### お題

`exp = 12` のときに `上級` と表示されるように直してください。

#### ヒント

大きい条件を先に書きます。

#### 確認ポイント

✅️ 12のとき `上級` になる
✅️ 条件の順番が正しくなっている
