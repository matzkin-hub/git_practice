## 「接触判定！当たり判定魔法を実装せよ」

## ミッション：プレイヤーを動かして、コインに触れたら回収できるようにしよう！

この課題では「要素の位置を取得する」「2つの四角が重なっているか判定する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">矢印キーで動いてコインを集めよう</p>
<p class="score">スコア：<span id="scoreDisplay">0</span></p>

<div id="stage">
  <div id="player">🧙</div>
  <div id="coin">🪙</div>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
```

JSタブに下のコードを貼り付けてください。

```js
const player = document.querySelector('#player');
const coin = document.querySelector('#coin');
const scoreDisplay = document.querySelector('#scoreDisplay');

let x = 20;
let y = 20;
let score = 0;

function renderPlayer() {
  player.style.left = x + 'px';
  player.style.top = y + 'px';
}

function moveCoin() {
  const coinX = Math.floor(Math.random() * 7) * 40;
  const coinY = Math.floor(Math.random() * 7) * 40;
  coin.style.left = coinX + 'px';
  coin.style.top = coinY + 'px';
}

function checkCollision() {
  const playerRect = player.getBoundingClientRect();
  const coinRect = coin.getBoundingClientRect();

  const isHit =
    playerRect.left < coinRect.right &&
    playerRect.right > coinRect.left &&
    playerRect.top < coinRect.bottom &&
    playerRect.bottom > coinRect.top;

  if (isHit) {
    score = score + 1;
    scoreDisplay.textContent = score;
    moveCoin();
  }
}

renderPlayer();
moveCoin();

document.addEventListener('keydown', function(event) {
  switch (event.key) {
    case 'ArrowUp':
      y = y - 40;
      break;
    case 'ArrowDown':
      y = y + 40;
      break;
    case 'ArrowLeft':
      x = x - 40;
      break;
    case 'ArrowRight':
      x = x + 40;
      break;
    default:
      return;
  }

  event.preventDefault();

  if (x < 0) {
    x = 0;
  }
  if (x > 280) {
    x = 280;
  }
  if (y < 0) {
    y = 0;
  }
  if (y > 280) {
    y = 280;
  }

  renderPlayer();
  checkCollision();
});
```

プレイヤーがコインに触れると、スコアが増えてコインの場所が変わることを確認してください。

### コードを読み解こう

#### `getBoundingClientRect()` ：要素の位置と大きさを取る

```js
const playerRect = player.getBoundingClientRect();
const coinRect = coin.getBoundingClientRect();
```

このメソッドを使うと、要素の左・右・上・下の位置をまとめて取れます。

#### 四角が重なっているかを調べる

```js
const isHit =
  playerRect.left < coinRect.right &&
  playerRect.right > coinRect.left &&
  playerRect.top < coinRect.bottom &&
  playerRect.bottom > coinRect.top;
```

少し長いですが、「左右も上下も重なっているなら当たっている」という意味です。

#### 当たったら次の処理をする

```js
if (isHit) {
  score = score + 1;
  moveCoin();
}
```

当たり判定は、それ自体がゴールではありません。重なったと分かったら、得点を増やしたり、アイテムを移動したりします。

#### グリッドに合わせてコインを置く

```js
const coinX = Math.floor(Math.random() * 7) * 40;
```

`40` ずつのマスにそろえておくと、プレイヤーと位置が合いやすくなります。

### 改造しよう

#### ミッション①：障害物を追加しよう

特定のマスには入れないようにすると、ゲームらしくなります。

#### ミッション②：コインの数を増やそう

2枚や3枚にすると、当たり判定を複数回使う練習になります。

#### ミッション③：5枚集めたらクリアにしよう

`if (score === 5)` のような条件を使って、ゲーム終了メッセージを出してみましょう。
