## 「残光演出！軌跡と発光で磨き上げよ」

## ミッション：花火の粒に軌跡と光のにじみを追加して、見た目を磨こう！

この課題では「履歴を配列で持つ」「半透明で画面を塗り直す」「グラデーションで発光を作る」技術を習得します。

動きはできたので、ここでは見た目を一段上げます。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">軌跡と光で花火をもっときれいにしよう</p>
<canvas id="canvas"></canvas>
```

CSSタブに下のコードを貼り付けてください。

```css

  display: block;
  background: linear-gradient(to bottom, #000428, #004e92);

```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

const rockets = [];
const particles = [];

const colorPalettes = [
  ['red', 'tomato', 'salmon', 'mistyrose'],
  ['lime', 'limegreen', 'lightgreen', 'honeydew'],
  ['blue', 'royalblue', 'lightskyblue', 'lavender'],
  ['yellow', 'gold', 'khaki', 'lightyellow'],
  ['magenta', 'orchid', 'violet', 'plum'],
  ['cyan', 'deepskyblue', 'paleturquoise', 'lightcyan']
];

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

function explode(x, y, palette) {
  for (let i = 0; i < 70; i++) {
    const angle = Math.random() * Math.PI * 2;
    const speed = Math.random() * 5 + 2;
    const color = palette[Math.floor(Math.random() * palette.length)];

    particles.push({
      x: x,
      y: y,
      vx: Math.cos(angle) * speed,
      vy: Math.sin(angle) * speed,
      gravity: 0.05,
      friction: 0.98,
      alpha: 1,
      size: Math.random() * 2 + 1,
      color: color,
      trail: [],
      trailLength: 5
    });
  }
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const targetY = event.clientY - rect.top;
  const palette = colorPalettes[Math.floor(Math.random() * colorPalettes.length)];

  rockets.push({
    x: x,
    y: canvas.height,
    targetY: targetY,
    vy: -8,
    palette: palette
  });
});

function animate() {
  ctx.fillStyle = 'rgba(0, 4, 40, 0.15)';
  ctx.fillRect(0, 0, canvas.width, canvas.height);

  for (let i = rockets.length - 1; i >= 0; i--) {
    const rocket = rockets[i];
    rocket.y = rocket.y + rocket.vy;

    ctx.beginPath();
    ctx.arc(rocket.x, rocket.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'white';
    ctx.fill();

    ctx.beginPath();
    ctx.moveTo(rocket.x, rocket.y);
    ctx.lineTo(rocket.x, rocket.y + 10);
    ctx.strokeStyle = 'rgba(255, 255, 255, 0.5)';
    ctx.stroke();

    if (rocket.y <= rocket.targetY) {
      explode(rocket.x, rocket.y, rocket.palette);
      rockets.splice(i, 1);
    }
  }

  for (let i = particles.length - 1; i >= 0; i--) {
    const particle = particles[i];

    particle.trail.push({ x: particle.x, y: particle.y });
    if (particle.trail.length > particle.trailLength) {
      particle.trail.shift();
    }

    particle.vy = particle.vy + particle.gravity;
    particle.vx = particle.vx * particle.friction;
    particle.vy = particle.vy * particle.friction;
    particle.x = particle.x + particle.vx;
    particle.y = particle.y + particle.vy;
    particle.alpha = particle.alpha - 0.01;

    for (let j = 0; j < particle.trail.length; j++) {
      const point = particle.trail[j];
      const alpha = (j / particle.trail.length) * particle.alpha;

      ctx.beginPath();
      ctx.arc(point.x, point.y, particle.size * 0.5, 0, Math.PI * 2);
      ctx.fillStyle = particle.color;
      ctx.globalAlpha = alpha * 0.5;
      ctx.fill();
    }

    ctx.beginPath();
    ctx.arc(particle.x, particle.y, particle.size, 0, Math.PI * 2);
    ctx.fillStyle = particle.color;
    ctx.globalAlpha = particle.alpha;
    ctx.fill();

    const gradient = ctx.createRadialGradient(
      particle.x,
      particle.y,
      0,
      particle.x,
      particle.y,
      particle.size * 3
    );
    gradient.addColorStop(0, particle.color);
    gradient.addColorStop(1, 'transparent');

    ctx.beginPath();
    ctx.arc(particle.x, particle.y, particle.size * 3, 0, Math.PI * 2);
    ctx.fillStyle = gradient;
    ctx.globalAlpha = particle.alpha * 0.3;
    ctx.fill();

    ctx.globalAlpha = 1;

    if (particle.alpha <= 0) {
      particles.splice(i, 1);
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

花火の粒に残像が付き、少し光って見えることを確認してください。

### コードを読み解こう

#### 半透明で背景を塗ると、前のフレームが少し残る

```js
ctx.fillStyle = 'rgba(0, 4, 40, 0.15)';
ctx.fillRect(0, 0, canvas.width, canvas.height);
```

完全な不透明ではなく半透明で塗ると、前のフレームの絵が少し残ります。これが残像の土台です。

#### `trail` ：過去の位置を覚える

```js
trail: [],
trailLength: 5
```

粒が通った位置を配列に入れておき、古いものから順に消しています。

#### 履歴を小さく薄く描く

```js
const alpha = (j / particle.trail.length) * particle.alpha;
```

古い位置ほど薄く描くと、尾のように見えます。

#### `createRadialGradient()` ：中心から外へ広がる光

```js
const gradient = ctx.createRadialGradient(...);
```

中心が濃く、外側が透明になるグラデーションを使うと、光っている感じを作れます。

### 改造しよう

#### ミッション①：残像の長さを変えよう

```js
trailLength: 5
```

数字を大きくすると、尾が長くなります。

#### ミッション②：残像の消え方を変えよう

```js
ctx.fillStyle = 'rgba(0, 4, 40, 0.15)';
```

最後の `0.15` を変えると、残像の残り方が変わります。

#### ミッション③：光の広がりを変えよう

```js
particle.size * 3
```

大きくすると、光のにじみが広がります。

✅️ 粒の履歴を配列で持っている
✅️ 軌跡が尾のように見える
✅️ `createRadialGradient()` で発光を作っている
