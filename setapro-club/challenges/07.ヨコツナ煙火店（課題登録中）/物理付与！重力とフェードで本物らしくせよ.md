## 「物理付与！重力とフェードで本物らしくせよ」

## ミッション：花火の粒に重力と減速とフェードアウトを加えて、自然な動きにしよう！

この課題では「速度に重力を足す」「少しずつ減速させる」「透明度で消えていく表現を作る」技術を習得します。

ここから、花火が一気に本物らしくなります。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">爆発した粒を自然に落とそう</p>
<canvas id="canvas"></canvas>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  overflow: hidden;
  background: #000;
  font-family: sans-serif;
}

.hint {
  position: absolute;
  top: 16px;
  left: 16px;
  margin: 0;
  color: white;
  z-index: 1;
}

canvas {
  display: block;
}
```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

const rockets = [];
const particles = [];

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

function drawSky() {
  ctx.fillStyle = '#06142b';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
}

function explode(x, y) {
  for (let i = 0; i < 60; i++) {
    const angle = Math.random() * Math.PI * 2;
    const speed = Math.random() * 5 + 2;

    particles.push({
      x: x,
      y: y,
      vx: Math.cos(angle) * speed,
      vy: Math.sin(angle) * speed,
      gravity: 0.05,
      friction: 0.98,
      alpha: 1,
      size: 2
    });
  }
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const targetY = event.clientY - rect.top;

  rockets.push({
    x: x,
    y: canvas.height,
    targetY: targetY,
    vy: -8
  });
});

function animate() {
  drawSky();

  for (let i = rockets.length - 1; i >= 0; i--) {
    const rocket = rockets[i];
    rocket.y = rocket.y + rocket.vy;

    ctx.beginPath();
    ctx.arc(rocket.x, rocket.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'white';
    ctx.fill();

    if (rocket.y <= rocket.targetY) {
      explode(rocket.x, rocket.y);
      rockets.splice(i, 1);
    }
  }

  for (let i = particles.length - 1; i >= 0; i--) {
    const particle = particles[i];

    particle.vy = particle.vy + particle.gravity;
    particle.vx = particle.vx * particle.friction;
    particle.vy = particle.vy * particle.friction;

    particle.x = particle.x + particle.vx;
    particle.y = particle.y + particle.vy;
    particle.alpha = particle.alpha - 0.015;

    ctx.beginPath();
    ctx.arc(particle.x, particle.y, particle.size, 0, Math.PI * 2);
    ctx.fillStyle = 'gold';
    ctx.globalAlpha = particle.alpha;
    ctx.fill();
    ctx.globalAlpha = 1;

    if (particle.alpha <= 0) {
      particles.splice(i, 1);
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

粒が少しずつ落ちながら減速し、最後に消えていくことを確認してください。

### コードを読み解こう

#### `gravity` ：下へ引っ張る力

```js
particle.vy = particle.vy + particle.gravity;
```

毎フレーム少しずつ `vy` を増やすと、粒が下に引っ張られていきます。

Canvasでは `y` が大きくなるほど下なので、`vy` が増えると落ちます。

#### `friction` ：少しずつ減速させる

```js
particle.vx = particle.vx * particle.friction;
particle.vy = particle.vy * particle.friction;
```

`0.98` のような1より少し小さい数を掛けると、速度が少しずつ小さくなります。

#### `alpha` ：透明度

```js
particle.alpha = particle.alpha - 0.015;
```

`alpha` は1で完全に見えて、0で完全に見えなくなります。

#### `ctx.globalAlpha` ：そのときの描画を透明にする

```js
ctx.globalAlpha = particle.alpha;
ctx.fill();
ctx.globalAlpha = 1;
```

描いたあとに `1` に戻しておかないと、次に描くものまで透明になってしまいます。

### 改造しよう

#### ミッション①：重力を強くしよう

```js
gravity: 0.05
```

数字を大きくすると、粒が早く落ちます。

#### ミッション②：消える速さを変えよう

```js
particle.alpha = particle.alpha - 0.015;
```

小さくすると長く残り、大きくするとすぐ消えます。

#### ミッション③：粒のサイズを変えよう

```js
size: 2
```

花火の印象がかなり変わります。


#### チェックしてみよう

✅️ 粒に重力を加えている
✅️ `friction` で減速させている
✅️ `alpha` でフェードアウトしている
✅️ `ctx.globalAlpha` を使っている
