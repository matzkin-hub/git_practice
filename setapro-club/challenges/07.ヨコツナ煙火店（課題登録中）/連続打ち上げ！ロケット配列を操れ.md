## 「連続打ち上げ！ロケット配列を操れ」

## ミッション：何回クリックしても、そのたびにロケットを打ち上げられるようにしよう！

この課題では「配列で複数のロケットを管理する」「逆順ループで消しながら更新する」技術を習得します。

ここから、花火が1発だけでなく何発も打ち上がるようになります。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">クリックした高さまでロケットを打ち上げよう</p>
<canvas id="canvas"></canvas>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  overflow: hidden;
  background: #000;
  font-family: sans-serif;
}

.hint {
  position: absolute;
  top: 16px;
  left: 16px;
  margin: 0;
  color: white;
  z-index: 1;
}

canvas {
  display: block;
}
```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

const rockets = [];

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

function drawSky() {
  ctx.fillStyle = '#06142b';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const targetY = event.clientY - rect.top;

  rockets.push({
    x: x,
    y: canvas.height,
    targetY: targetY,
    vy: -8
  });
});

function animate() {
  drawSky();

  for (let i = rockets.length - 1; i >= 0; i--) {
    const rocket = rockets[i];

    rocket.y = rocket.y + rocket.vy;

    ctx.beginPath();
    ctx.arc(rocket.x, rocket.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'white';
    ctx.fill();

    ctx.beginPath();
    ctx.moveTo(rocket.x, rocket.y);
    ctx.lineTo(rocket.x, rocket.y + 10);
    ctx.strokeStyle = 'rgba(255, 255, 255, 0.5)';
    ctx.lineWidth = 1;
    ctx.stroke();

    if (rocket.y <= rocket.targetY) {
      rockets.splice(i, 1);
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

何回クリックしても、そのたびにロケットが飛ぶことを確認してください。

### コードを読み解こう

#### `const rockets = []` ：ロケット一覧を持つ

```js
const rockets = [];
```

ロケットが1個だけなら変数1つで持てますが、花火では何発も同時に飛びます。そのため、配列にまとめて入れています。

#### クリックのたびに `push()` で追加する

```js
rockets.push({
  x: x,
  y: canvas.height,
  targetY: targetY,
  vy: -8
});
```

クリックするたびに、新しいロケットオブジェクトを配列の最後に追加しています。

#### `targetY` ：どこで止まるかを持たせる

```js
targetY: targetY
```

クリックした高さまで飛ばしたいので、ロケットごとに「目標の高さ」を覚えています。

#### 逆順ループで削除する

```js
for (let i = rockets.length - 1; i >= 0; i--) {
```

配列をループしながら `splice()` で消すときは、後ろから見ていくのが安全です。

前から消すと、次の要素がずれて見落とすことがあります。

### 改造しよう

#### ミッション①：ロケットの速さを変えよう

```js
vy: -8
```

数字を変えると、打ち上がる速さが変わります。

#### ミッション②：軌跡を長くしよう

```js
ctx.lineTo(rocket.x, rocket.y + 10);
```

`10` を大きくすると、尾が長くなります。

#### ミッション③：ロケットの色を変えよう

```js
ctx.fillStyle = 'white';
```

黄色や赤にすると、見た目の印象が変わります。


#### チェックしてみよう

✅️ 配列でロケットを管理している
✅️ クリックするたびに新しいロケットが追加される
✅️ ロケットごとに `targetY` を持っている
✅️ 逆順ループでロケットを削除している
