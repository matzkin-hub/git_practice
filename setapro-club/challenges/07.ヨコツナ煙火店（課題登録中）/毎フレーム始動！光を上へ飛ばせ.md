## 「毎フレーム始動！光を上へ飛ばせ」

## ミッション：クリックした位置に向かって、下から光が飛び上がるようにしよう！

この課題では「`requestAnimationFrame()` で毎フレーム処理する」「位置を少しずつ更新して動きを作る」技術を習得します。

花火のロケットは、1回描いて終わりではなく、毎フレーム少しずつ動かして作ります。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">クリックすると光が下から飛びます</p>
<canvas id="canvas"></canvas>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  overflow: hidden;
  background: #000;
  font-family: sans-serif;
}

.hint {
  position: absolute;
  top: 16px;
  left: 16px;
  margin: 0;
  color: white;
  z-index: 1;
}

canvas {
  display: block;
}
```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

let light = null;

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

function drawSky() {
  ctx.fillStyle = '#06142b';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;

  light = {
    x: x,
    y: canvas.height - 20,
    vy: -6
  };
});

function animate() {
  drawSky();

  if (light !== null) {
    light.y = light.y + light.vy;

    ctx.beginPath();
    ctx.arc(light.x, light.y, 5, 0, Math.PI * 2);
    ctx.fillStyle = 'gold';
    ctx.fill();

    if (light.y < -10) {
      light = null;
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

クリックすると、画面の下から上へ光が飛ぶことを確認してください。

### コードを読み解こう

#### `requestAnimationFrame()` ：次の描画タイミングでまた呼ぶ

```js
requestAnimationFrame(animate);
```

この1行で、`animate()` が何度も呼ばれ続けます。これがアニメーションの基本です。

#### 毎フレーム背景を描き直す

```js
drawSky();
```

前のフレームの絵を残したままだと、光が移動した跡が全部残ります。今回はまず背景を描き直してから、新しい位置に光を描いています。

#### 位置を少しずつ変えると動いて見える

```js
light.y = light.y + light.vy;
```

`vy` は縦方向の速度です。今回は `-6` なので、`y` が毎回小さくなり、上へ動いていきます。

#### 1回のクリックで1個だけ動かす

```js
let light = null;
```

今回はまだ超入門なので、動く光は1個だけです。次の回で、これを複数同時に動かせるようにします。

### 改造しよう

#### ミッション①：光の速さを変えよう

```js
vy: -6
```

数字を大きくすると速く、小さくするとゆっくり飛びます。

#### ミッション②：光の大きさを変えよう

```js
ctx.arc(light.x, light.y, 5, 0, Math.PI * 2);
```

半径を変えると見た目が変わります。

#### ミッション③：色を変えよう

```js
ctx.fillStyle = 'gold';
```

白い光にすると、ロケットらしく見えます。


#### チェックしてみよう

✅️ `requestAnimationFrame()` を使っている
✅️ クリックすると下から光が飛ぶ
✅️ `y` の値を毎フレーム更新している
✅️ 背景を描き直してアニメーションにしている
