## 「座標捕捉！クリック位置に光点を描け」

## ミッション：Canvasをクリックした場所に、小さな光の点を描こう！

この課題では「クリック位置をCanvas座標に変換する」「`arc()` で丸を描く」技術を習得します。

花火を打ち上げるには、まず「どこをクリックしたか」を正しく取れる必要があります。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">Canvasをクリックして光点を出してみよう</p>
<canvas id="canvas"></canvas>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  overflow: hidden;
  background: #000;
  font-family: sans-serif;
}

.hint {
  position: absolute;
  top: 16px;
  left: 16px;
  margin: 0;
  color: white;
  z-index: 1;
}

canvas {
  display: block;
}
```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

function drawSky() {
  ctx.fillStyle = '#06142b';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
}

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
  drawSky();
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const y = event.clientY - rect.top;

  ctx.beginPath();
  ctx.arc(x, y, 6, 0, Math.PI * 2);
  ctx.fillStyle = 'gold';
  ctx.fill();
});
```

クリックした場所に、黄色い光点が出ることを確認してください。

### コードを読み解こう

#### `getBoundingClientRect()` ：Canvasの位置を調べる

```js
const rect = canvas.getBoundingClientRect();
```

`event.clientX` と `event.clientY` は、ブラウザ画面全体の中での座標です。

でも、欲しいのはCanvasの左上を基準にした座標です。そのため、まずCanvasが画面のどこにあるかを `getBoundingClientRect()` で調べます。

#### Canvas用の座標に変換する

```js
const x = event.clientX - rect.left;
const y = event.clientY - rect.top;
```

Canvasの左端と上端を引くことで、「Canvasの中ではどこか」が分かります。

#### `beginPath()` と `arc()` ：丸を描く

```js
ctx.beginPath();
ctx.arc(x, y, 6, 0, Math.PI * 2);
ctx.fill();
```

`arc()` は円や丸を描くための命令です。

| 引数 | 意味 |
|------|------|
| `x` | 丸の中心のX座標 |
| `y` | 丸の中心のY座標 |
| `6` | 半径 |
| `0` | 開始角度 |
| `Math.PI * 2` | 一周分の角度 |

`Math.PI * 2` は、1周の円を表します。

#### Canvasは前に描いたものが残る

今回はクリックするたびに背景を消していないので、光点がどんどん増えていきます。

これもCanvasの大事な特徴です。

### 改造しよう

#### ミッション①：光点の色を変えよう

```js
ctx.fillStyle = 'gold';
```

白や水色に変えると、星のように見えます。

#### ミッション②：サイズを変えよう

```js
ctx.arc(x, y, 6, 0, Math.PI * 2);
```

`6` を変えると、光点の大きさが変わります。

#### ミッション③：背景を塗り直すボタンを追加しよう

`drawSky()` をもう一度呼ぶと、Canvasをリセットできます。


#### チェックしてみよう

✅️ クリックした場所に光点が出る
✅️ `getBoundingClientRect()` を使っている
✅️ `event.clientX` と `event.clientY` を使っている
✅️ `arc()` で丸を描いている
