## 「爆裂解放！粒子を放射状にばらまけ」

## ミッション：ロケットが目標に着いたら、その場所で花火の粒を爆発させよう！

この課題では「1つの中心から大量の粒を作る」「角度と速度を使って放射状に広げる」技術を習得します。

ここで、ただのロケットが「花火」になります。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">クリックした場所で花火を爆発させよう</p>
<canvas id="canvas"></canvas>
```

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

const rockets = [];
const particles = [];

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

function drawSky() {
  ctx.fillStyle = '#06142b';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
}

function explode(x, y) {
  for (let i = 0; i < 50; i++) {
    const angle = Math.random() * Math.PI * 2;
    const speed = Math.random() * 4 + 2;

    particles.push({
      x: x,
      y: y,
      vx: Math.cos(angle) * speed,
      vy: Math.sin(angle) * speed,
      life: 40
    });
  }
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const targetY = event.clientY - rect.top;

  rockets.push({
    x: x,
    y: canvas.height,
    targetY: targetY,
    vy: -8
  });
});

function animate() {
  drawSky();

  for (let i = rockets.length - 1; i >= 0; i--) {
    const rocket = rockets[i];
    rocket.y = rocket.y + rocket.vy;

    ctx.beginPath();
    ctx.arc(rocket.x, rocket.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'white';
    ctx.fill();

    if (rocket.y <= rocket.targetY) {
      explode(rocket.x, rocket.y);
      rockets.splice(i, 1);
    }
  }

  for (let i = particles.length - 1; i >= 0; i--) {
    const particle = particles[i];

    particle.x = particle.x + particle.vx;
    particle.y = particle.y + particle.vy;
    particle.life = particle.life - 1;

    ctx.beginPath();
    ctx.arc(particle.x, particle.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'gold';
    ctx.fill();

    if (particle.life <= 0) {
      particles.splice(i, 1);
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

ロケットが上まで飛んだあと、その場所で粒が放射状に広がることを確認してください。

### コードを読み解こう

#### `explode(x, y)` ：爆発を1つの関数にまとめる

```js
function explode(x, y) {
  ...
}
```

爆発の処理は長くなりやすいので、関数にまとめると読みやすくなります。

#### 角度と速度で方向を決める

```js
const angle = Math.random() * Math.PI * 2;
const speed = Math.random() * 4 + 2;
```

`angle` は向き、`speed` は速さです。

#### `Math.cos()` と `Math.sin()` ：向きをXとYに分ける

```js
vx: Math.cos(angle) * speed,
vy: Math.sin(angle) * speed,
```

向きだけでは移動できないので、横方向の速さ `vx` と縦方向の速さ `vy` に分けています。

これで粒が360度に広がります。

#### `life` ：粒の寿命

```js
life: 40
```

粒はずっと残ると困るので、寿命を持たせています。毎フレーム減らして、0になったら削除しています。
