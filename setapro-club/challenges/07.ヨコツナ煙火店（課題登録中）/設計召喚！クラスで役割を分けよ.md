## 「設計召喚！クラスで役割を分けよ」

## ミッション：ロケットと粒をクラスに分けて、コードを整理しよう！

この課題では「`class` で設計図を作る」「`update()` と `draw()` に責務を分ける」技術を習得します。

機能が増えてきたので、ここで一度きれいに整理します。

JSタブに下のコードを貼り付けてください。

```js
const canvas = document.querySelector('#canvas');
const ctx = canvas.getContext('2d');

const colorPalettes = [
  ['red', 'tomato', 'salmon', 'mistyrose'],
  ['lime', 'limegreen', 'lightgreen', 'honeydew'],
  ['blue', 'royalblue', 'lightskyblue', 'lavender'],
  ['yellow', 'gold', 'khaki', 'lightyellow'],
  ['magenta', 'orchid', 'violet', 'plum'],
  ['cyan', 'deepskyblue', 'paleturquoise', 'lightcyan']
];

const rockets = [];
const particles = [];

class Particle {
  constructor(x, y, color) {
    this.x = x;
    this.y = y;
    this.color = color;

    const angle = Math.random() * Math.PI * 2;
    const speed = Math.random() * 5 + 2;
    this.vx = Math.cos(angle) * speed;
    this.vy = Math.sin(angle) * speed;

    this.gravity = 0.05;
    this.friction = 0.98;
    this.alpha = 1;
    this.size = Math.random() * 2 + 1;
  }

  update() {
    this.vy = this.vy + this.gravity;
    this.vx = this.vx * this.friction;
    this.vy = this.vy * this.friction;
    this.x = this.x + this.vx;
    this.y = this.y + this.vy;
    this.alpha = this.alpha - 0.015;
  }

  draw() {
    ctx.beginPath();
    ctx.arc(this.x, this.y, this.size, 0, Math.PI * 2);
    ctx.fillStyle = this.color;
    ctx.globalAlpha = this.alpha;
    ctx.fill();
    ctx.globalAlpha = 1;
  }

  isDead() {
    return this.alpha <= 0;
  }
}

class Rocket {
  constructor(x, targetY) {
    this.x = x;
    this.y = canvas.height;
    this.targetY = targetY;
    this.vy = -8;
    this.exploded = false;
    this.palette = colorPalettes[Math.floor(Math.random() * colorPalettes.length)];
  }

  update() {
    this.y = this.y + this.vy;

    if (this.y <= this.targetY) {
      this.explode();
    }
  }

  draw() {
    ctx.beginPath();
    ctx.arc(this.x, this.y, 2, 0, Math.PI * 2);
    ctx.fillStyle = 'white';
    ctx.fill();

    ctx.beginPath();
    ctx.moveTo(this.x, this.y);
    ctx.lineTo(this.x, this.y + 10);
    ctx.strokeStyle = 'rgba(255, 255, 255, 0.5)';
    ctx.stroke();
  }

  explode() {
    this.exploded = true;

    for (let i = 0; i < 80; i++) {
      const color = this.palette[Math.floor(Math.random() * this.palette.length)];
      particles.push(new Particle(this.x, this.y, color));
    }
  }
}

function resizeCanvas() {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
}

resizeCanvas();

window.addEventListener('resize', function() {
  resizeCanvas();
});

canvas.addEventListener('click', function(event) {
  const rect = canvas.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const targetY = event.clientY - rect.top;

  rockets.push(new Rocket(x, targetY));
});

function animate() {
  ctx.fillStyle = 'rgba(0, 4, 40, 0.2)';
  ctx.fillRect(0, 0, canvas.width, canvas.height);

  for (let i = rockets.length - 1; i >= 0; i--) {
    rockets[i].update();
    rockets[i].draw();

    if (rockets[i].exploded) {
      rockets.splice(i, 1);
    }
  }

  for (let i = particles.length - 1; i >= 0; i--) {
    particles[i].update();
    particles[i].draw();

    if (particles[i].isDead()) {
      particles.splice(i, 1);
    }
  }

  requestAnimationFrame(animate);
}

animate();
```

クリックすると、整理されたコードのまま花火が動くことを確認してください。

### コードを読み解こう

#### `class Particle` と `class Rocket`

```js
class Particle {
  ...
}

class Rocket {
  ...
}
```

粒とロケットは役割が違います。別々のクラスにすると、「どのデータを持つか」「どう動くか」がはっきりします。

#### `constructor` ：最初の状態を作る

```js
constructor(x, targetY) {
  this.x = x;
  this.y = canvas.height;
}
```

新しく作られた瞬間の状態を決めています。

#### `update()` と `draw()` を分ける

```js
rockets[i].update();
rockets[i].draw();
```

動きを計算する処理と、見た目を描く処理を分けると読みやすくなります。

#### `isDead()` ：消える条件もオブジェクト自身に持たせる

```js
if (particles[i].isDead()) {
```

「いつ消えるか」を呼び出し側で毎回書くより、粒自身に判定させる方が整理しやすいです。
