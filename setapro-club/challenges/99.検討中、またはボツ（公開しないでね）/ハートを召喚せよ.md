## 「ハートを召喚せよ」

**ミッション：募金ボタンを押すと、クリックした場所にハートが1個現れるようにしよう！**

前に作ったページにハートエフェクトを追加していきます。まず1個だけ出せるようにします。

### 前のコードからスタートする

前に作ったをそのまま使います。新しくコードを書く前にコードが入っていることを確認してください。

### コードを追加しよう

コードに追加・変更する内容だけを説明します。

#### CSS に追加する

既存のCSSの一番下に下のコードを追加してください。

```css
.heart {
  position: fixed;
  color: #e05080;
  font-size: 2rem;
  pointer-events: none;
  transform: translate(-50%, -50%);
}
```

#### JS を変更する

前の課題のJSを下のコードに書き換えてください。

```js
let count = 0;
const btn = document.querySelector('#donateBtn');
const countDisplay = document.querySelector('#countDisplay');

btn.addEventListener('click', function(event) {
  // カウントを増やす（F-01と同じ）
  count = count + 1;
  countDisplay.textContent = count;

  // ハートを1個出す（新しく追加）
  const heart = document.createElement('div');
  heart.textContent = '♥';
  heart.classList.add('heart');
  heart.style.left = event.clientX + 'px';
  heart.style.top  = event.clientY + 'px';
  document.body.appendChild(heart);
});
```

「♥ 募金する」ボタンを押すと、ボタンのクリック位置にハートが現れることを確認してください。

### コードを読み解こう

#### 何が変わったか

変わったのはJSだけです。

```
変更前：btn.addEventListener('click', function() {
変更後：btn.addEventListener('click', function(event) {
```

`function()` に `event` を追加しました。`event.clientX/Y` でクリック座標を受け取るために必要です。

#### ハートを出す3ステップ

```js
const heart = document.createElement('div');  // 1. ハート要素を工作
heart.textContent = '♥';                      // 2. ハート文字を入れる
heart.classList.add('heart');                  // 3. スタイルを着せる
heart.style.left = event.clientX + 'px';      // 4. X座標を指定
heart.style.top  = event.clientY + 'px';      // 5. Y座標を指定
document.body.appendChild(heart);             // 6. 画面に出す
```

円（丸）を出したのと同じ手順です。`createElement('div')` で箱を作り、`textContent = '♥'` でハート文字を入れています。

#### なぜ `position: fixed` + `transform: translate(-50%, -50%)` なのか

`position: fixed` で座標指定できるようにして、`transform: translate(-50%, -50%)` でハートの中心がクリック位置に来るように調整しています。

### 改造しよう

#### ミッション：ハートの文字やサイズを変えよう

```css
.heart {
  font-size: 2rem;  ← サイズを変える（大きくしたければ数字を増やす）
  color: #e05080;   ← 色を変える
}
```

```js
heart.textContent = '♥';  ← 別の文字や絵文字にしてもOK（例：'❤'、'★'、'🌸'）
```
