## 「ハートに飛翔の呪いを」

**ミッション：ハートが上に飛び上がりながらフェードアウトして消えるようにしよう！**

ここまではハートが出現するだけでした。
ここではアニメーションと自動削除を追加して、「飛んで消える」動きを作ります。

### 前の課題のコードからスタートする

### CSS を変更する

前の `.heart` スタイルを下のコードに書き換えてください。

```css
.heart {
  position: fixed;
  color: #e05080;
  font-size: 2rem;
  pointer-events: none;
  transform: translate(-50%, -50%);
}

.heart.flying {
  animation: heartFly 1s ease-out forwards;
}

@keyframes heartFly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(-50%, calc(-50% - 100px));
  }
}
```

### JS を変更する

ハートを出す部分に2行追加します。

```js
let count = 0;
const btn = document.querySelector('#donateBtn');
const countDisplay = document.querySelector('#countDisplay');

btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;

  const heart = document.createElement('div');
  heart.textContent = '♥';
  heart.classList.add('heart');
  heart.style.left = event.clientX + 'px';
  heart.style.top  = event.clientY + 'px';
  document.body.appendChild(heart);

  heart.classList.add('flying');  // 追加：アニメーション開始

  setTimeout(function() {        // 追加：1秒後に削除
    heart.remove();
  }, 1000);
});
```

ボタンを押すとハートが上に飛んでフェードアウトして消えることを確認してください。

### コードを読み解こう

#### `.heart.flying`：クラスを組み合わせる


```css
.heart.flying {
  animation: heartFly 1s ease-out forwards;
}
```

`.heart.flying` は「`heart` クラスと `flying` クラスの両方が付いている要素」に適用されるCSSです。

最初は `heart` クラスだけ付けます。`flying` クラスはあとからJSで追加します。

```js
heart.classList.add('heart');    // まず heart クラスだけ付ける
// ...
heart.classList.add('flying');   // あとで flying クラスも付ける → アニメーション開始
```

#### `@keyframes heartFly`：2つの値を同時にアニメーションさせる

```css
@keyframes heartFly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(-50%, calc(-50% - 100px));
  }
}
```

`opacity` と `transform` を同時に変化させています。

| プロパティ | from（開始） | to（終了） | 効果 |
|-----------|------------|---------|------|
| `opacity` | `1`（見える） | `0`（見えない） | フェードアウト |
| `transform` | `-50%, -50%` | `-50%, calc(-50% - 100px)` | 100px上へ移動 |

`opacity: 0` になっても要素はまだDOMに残っています。`setTimeout` で `remove()` を呼ぶまでは存在し続けますが、見えないだけです。

#### アニメーション時間と `setTimeout` を合わせる

```css
animation: heartFly 1s ease-out forwards;
                    ↑
                   1秒
```

```js
setTimeout(function() {
  heart.remove();
}, 1000);  ← 1000ミリ秒 = 1秒
```

アニメーション時間（`1s`）と `setTimeout` の時間（`1000`ミリ秒）を合わせることで、アニメーションが終わった直後に要素が削除されます。

### 改造しよう

#### ミッション①：飛ぶ高さを変えよう

```css
transform: translate(-50%, calc(-50% - 100px));
                                         ↑ ここを変える
```


#### ミッション②：アニメーションのスピードを変えよう

`animation: heartFly 1s ...` の `1s` と `setTimeout` の `1000` を同じ割合で変えましょう。

例：`0.8s` にしたら `setTimeout` は `800` にします。


#### チェックしてみよう

✅️ ボタンを押すとハートが上に飛ぶ
✅️ ハートがフェードアウトして消える
✅️ `@keyframes` でアニメーションを定義している
✅️ `setTimeout` + `remove()` で要素を削除している
