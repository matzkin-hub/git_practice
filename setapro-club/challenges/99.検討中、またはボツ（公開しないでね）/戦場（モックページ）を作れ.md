## 「戦場（モックページ）を作れ」

**ミッション：ハートエフェクトを実装するクリック募金ページの土台をつくろう！**

第3章では、このページにハートが飛び散るエフェクトを追加していきます。まず大ボスとの戦場となるページを用意しましょう。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="page">
  <h1>クリック募金</h1>
  <p class="desc">ボタンを1回押すごとに、1円が寄付されます。</p>
  <button id="donateBtn">♥ 募金する</button>
  <p class="count">寄付回数：<span id="countDisplay">0</span>回</p>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #fff0f5;
  font-family: sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

.page {
  text-align: center;
  padding: 40px;
}

h1 {
  color: #e05080;
  font-size: 2rem;
  margin-bottom: 16px;
}

.desc {
  color: #666;
  margin-bottom: 32px;
}

#donateBtn {
  background: #e05080;
  color: white;
  border: none;
  padding: 20px 40px;
  font-size: 1.5rem;
  border-radius: 50px;
  cursor: pointer;
}

#donateBtn:hover {
  background: #c03060;
}

.count {
  color: #888;
  margin-top: 24px;
}
```

JSタブに下のコードを貼り付けてください。

```js
let count = 0;
const btn = document.querySelector('#donateBtn');
const countDisplay = document.querySelector('#countDisplay');

btn.addEventListener('click', function() {
  count = count + 1;
  countDisplay.textContent = count;
});
```

「♥ 募金する」ボタンを押すと寄付回数が増えることを確認してください。

### コードを読み解こう

#### HTMLの構造：ページの骨格

```html
<div class="page">        ← ページ全体を包む箱
  <h1>...</h1>            ← タイトル
  <p class="desc">...</p> ← 説明文
  <button id="donateBtn"> ← 募金ボタン（JSで操作する）
  <p class="count">
    <span id="countDisplay">0</span>  ← カウント表示（JSで書き換える）
  </p>
</div>
```

`<span>` は `<p>` と似ていますが、文章の途中の一部分だけを囲むのに使います。「寄付回数：0回」の数字部分だけを `<span>` で囲むことで、数字だけをJavaScriptで書き換えられます。

#### CSSのポイント：ページを中央に配置する

```css
body {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

`display: flex` + `justify-content: center` + `align-items: center` の組み合わせで、子要素（`.page`）を画面の中央に配置できます。

#### JSの仕組み

カウント表示は「数を刻む石板」と同じ仕組みです。`let count` で黒板を用意して、押すたびに1増やして画面に反映します。

### 改造しよう

#### ミッション：ページをカスタマイズしよう

第3章の大ボスでは「自分でカスタマイズした要素が1つ以上ある」が完成条件の一つです。今のうちに自分らしいデザインにしておきましょう。

変えられるもの：
- タイトルの文字（`<h1>` の中身）
- 説明文（`.desc` の中身）
- ボタンの文字（`<button>` の中身）
- 色（`#e05080` の部分）
- ボタンの形（`border-radius` の値）


#### チェックしてみよう

✅️ タイトル・説明文・ボタン・カウントが表示されている
✅️ ボタンを押すと寄付回数が増える
