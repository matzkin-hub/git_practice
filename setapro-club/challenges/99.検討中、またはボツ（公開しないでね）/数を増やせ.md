## 「数を増やせ」

**ミッション：ボタンを押すと、ハートが1個ではなく複数個飛び散るようにしよう！**

ここまではハートが1個だけ飛んでいました。
ここでは `for` ループを使って一度に複数のハートを生成します。

### JS を変更しよう

前のJSを下のコードに書き換えてください。変わったのはクリックイベントの中のハートを出す部分だけです。

```js
let count = 0;
const btn = document.querySelector('#donateBtn');
const countDisplay = document.querySelector('#countDisplay');

btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;

  for (let i = 0; i < 5; i++) {
    const heart = document.createElement('div');
    heart.textContent = '♥';
    heart.classList.add('heart');
    heart.style.left = event.clientX + 'px';
    heart.style.top  = event.clientY + 'px';
    document.body.appendChild(heart);

    heart.classList.add('flying');

    setTimeout(function() {
      heart.remove();
    }, 1000);
  }
});
```

ボタンを押すと5個のハートが同じ場所から飛び上がることを確認してください。

### コードを読み解こう

#### `for (let i = 0; i < 5; i++)`：同じ処理を5回繰り返す

```js
for (let i = 0; i < 5; i++) {
  // この中のコードが5回実行される
}
```

`for` ループは「同じ処理を指定した回数繰り返す」命令です。

| 部分 | 意味 |
|------|------|
| `let i = 0` | カウンター変数 `i` を 0 から始める |
| `i < 5` | `i` が 5 より小さい間は繰り返す（0, 1, 2, 3, 4 の5回） |
| `i++` | 1回処理するたびに `i` を1増やす（`i = i + 1` と同じ） |

**繰り返しの流れ：**

```
i = 0 → ハートを1個出す
i = 1 → ハートを1個出す
i = 2 → ハートを1個出す
i = 3 → ハートを1個出す
i = 4 → ハートを1個出す
i = 5 → 「i < 5」を満たさないので終了
合計5個のハートが出る
```

#### 今の状態：5個が全員同じ場所から同じ方向に飛ぶ

今のコードでは全員が `event.clientX`・`event.clientY` にまったく同じ座標を持っています。同じCSSアニメーションも適用されるので、5個全員が完全に重なって同じ動きをします。

これを次の「ランダムに散らせ」でランダムな座標オフセットと方向を加えて「バラバラに散らす」ようにします。
