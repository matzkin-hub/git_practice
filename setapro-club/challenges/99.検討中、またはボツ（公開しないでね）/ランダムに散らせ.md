## 「ランダムに散らせ」

**ミッション：ハートの方向・サイズ・色をバラバラにして、本物のエフェクトらしくしよう！**

ここまでは複数のハートが同じ場所から同じ方向に飛んでいました。
ここでは `Math.random()` を使って、それぞれのハートが違う方向・サイズ・色で飛ぶようにします。

### CSS を変更しよう

CSSにCSS変数（カスタムプロパティ）を追加して、ハートごとに飛ぶ方向を変えられるようにします。

`.heart.flying` と `@keyframes heartFly` の部分を下のコードに書き換えてください。

```css
.heart.flying {
  animation: heartFly 1s ease-out forwards;
}

@keyframes heartFly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(
      calc(-50% + var(--dx)),
      calc(-50% + var(--dy))
    );
  }
}
```

### JS を変更しよう

`for` ループの中のハートを出す部分を下のコードに書き換えてください。



```js
btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;

  for (let i = 0; i < 8; i++) {
    const heart = document.createElement('div');
    heart.textContent = '♥';
    heart.classList.add('heart');
    heart.style.left = event.clientX + 'px';
    heart.style.top  = event.clientY + 'px';

    // ランダムな飛ぶ方向（dx: 横, dy: 縦）
    const dx = (Math.random() - 0.5) * 120;
    const dy = -(Math.random() * 80 + 60);
    heart.style.setProperty('--dx', dx + 'px');
    heart.style.setProperty('--dy', dy + 'px');

    // ランダムなサイズ
    const size = Math.random() * 16 + 16;
    heart.style.fontSize = size + 'px';

    // ランダムな色
    const colors = ['#e05080', '#ff6b9d', '#ff4444', '#ff8c00', '#ff69b4'];
    heart.style.color = colors[Math.floor(Math.random() * colors.length)];

    document.body.appendChild(heart);
    heart.classList.add('flying');

    setTimeout(function() {
      heart.remove();
    }, 1000);
  }
});
```

ボタンを押すとハートがバラバラの方向に飛び散ることを確認してください。

### コードを読み解こう

#### CSS変数（`--dx`, `--dy`）：CSSに値を渡すスロット

```css
transform: translate(
  calc(-50% + var(--dx)),
  calc(-50% + var(--dy))
);
```

`var(--dx)` は「`--dx` というスロットに入っている値を使う」という意味です。スロットの値はJavaScriptから設定できます。

```js
heart.style.setProperty('--dx', dx + 'px');
heart.style.setProperty('--dy', dy + 'px');
```

`setProperty('--dx', ...)` で `--dx` スロットに値を入れます。ハートごとに違う値を入れることで、全員が違う方向に飛びます。

**CSS変数があることで実現できること：**

```
ハート1: --dx = -45px, --dy = -90px  → 左上方向に飛ぶ
ハート2: --dx = +30px, --dy = -110px → 右上方向に飛ぶ
ハート3: --dx = -10px, --dy = -70px  → ほぼ真上に飛ぶ
```

#### 方向の計算

```js
const dx = (Math.random() - 0.5) * 120;  // -60〜+60（横方向のズレ）
const dy = -(Math.random() * 80 + 60);   // -60〜-140（必ず上方向）
```

| 式 | 値の範囲 | 意味 |
|----|---------|------|
| `Math.random() - 0.5` | -0.5〜+0.5 | マイナスにもプラスにもなる |
| `× 120` | -60〜+60 | 横方向のズレ幅 |
| `-(Math.random() * 80 + 60)` | -60〜-140 | 必ず上に飛ぶ（マイナスが上方向） |

#### `font-size` でハートのサイズを変える

```js
const size = Math.random() * 16 + 16;
heart.style.fontSize = size + 'px';
```

ハートは絵文字・テキスト文字なので、`width/height` ではなく `font-size` でサイズが変わります。`16〜32px` の範囲でランダムなサイズになります。
