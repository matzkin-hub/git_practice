## 「大ボス：全魔法を合体させて決戦に挑め！」

## ミッション：ボタンを押すとハートが飛び散るいいね！エフェクトページを完成させよう！

ここまで習得した技術をすべて組み合わせて、本物のエフェクトを作ります。一度に全部書くのではなく、ステップごとに少しずつ機能を追加していきます。


## ステップ1：土台を作る

まずページの骨格とカウント機能を作ります。

HTMLタブに下のコードを貼り付けてください。

```html
<div class="page">
  <p class="post-label">📸 今日のごはん</p>
  <p class="desc">おいしそうと思ったらいいね！してね</p>
  <button id="likeBtn">♥ いいね！</button>
  <p class="count">♥ <span id="countDisplay">0</span></p>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #fff0f5;
  font-family: sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

.page {
  text-align: center;
  padding: 40px;
}

.post-label {
  font-size: 1.4rem;
  font-weight: bold;
  color: #333;
  margin-bottom: 8px;
}

.desc {
  color: #666;
  margin-bottom: 32px;
}

#likeBtn {
  background: #e05080;
  color: white;
  border: none;
  padding: 20px 40px;
  font-size: 1.5rem;
  border-radius: 50px;
  cursor: pointer;
}

#likeBtn:hover {
  background: #c03060;
}

.count {
  color: #888;
  margin-top: 24px;
}
```

JSタブに下のコードを貼り付けてください。

```js
let count = 0;
const btn = document.querySelector('#likeBtn');
const countDisplay = document.querySelector('#countDisplay');

btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;
});
```

:::note info
ボタンを押すといいね数が増えることを確認してください。
:::


## ステップ2：クリック位置にハートを1個出す

次に、ボタンを押した場所にハートが1個現れるようにします。

#### CSS

CSSの一番下に追加してください。

```css
.heart {
  position: fixed;
  color: #e05080;
  font-size: 2rem;
  pointer-events: none;
  transform: translate(-50%, -50%);
}
```

#### JS

`countDisplay.textContent = count;` の行の後に、以下を追加してください。

```js
const heart = document.createElement('div');
heart.textContent = '♥';
heart.classList.add('heart');
heart.style.left = event.clientX + 'px';
heart.style.top  = event.clientY + 'px';
document.body.appendChild(heart);
```

:::note info
ボタンを押すとクリック位置にハートが現れることを確認してください。（まだ消えません）
:::


## ステップ3：ハートを飛ばして消す

ハートが上に飛び上がりながらフェードアウトして消えるようにします。

#### CSS

CSSの `.heart { }` の下に追加してください。

```css
.heart.flying {
  animation: heartFly 1s ease-out forwards;
}

@keyframes heartFly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(-50%, calc(-50% - 100px));
  }
}
```

#### JS

`document.body.appendChild(heart);` の行の後に、以下を追加してください。

```js
heart.classList.add('flying');
setTimeout(function() {
  heart.remove();
}, 1000);
```

:::note info
ボタンを押すとハートが上に飛んでフェードアウトして消えることを確認してください。
:::


## ステップ4：ハートを複数個飛ばす

`for` ループを使って、1回のクリックで複数のハートを出します。

ハートを出す部分を `for` ループで囲むため、JSを下のコードに書き換えてください。

```js
btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;

  for (let i = 0; i < 8; i++) {
    const heart = document.createElement('div');
    heart.textContent = '♥';
    heart.classList.add('heart');
    heart.style.left = event.clientX + 'px';
    heart.style.top  = event.clientY + 'px';
    document.body.appendChild(heart);
    heart.classList.add('flying');
    setTimeout(function() {
      heart.remove();
    }, 1000);
  }
});
```

:::note info
ボタンを押すと8個のハートが飛ぶことを確認してください。（まだ全員同じ方向に飛びます）
:::


## ステップ5：ランダムに散らす

`Math.random()` とCSS変数を使って、方向・サイズ・色をバラバラにします。

#### CSS

`@keyframes heartFly` の `to { }` の中身を、以下に書き換えてください。

```css
to {
  opacity: 0;
  transform: translate(
    calc(-50% + var(--dx)),
    calc(-50% + var(--dy))
  );
}
```

#### JS

`heart.style.top = event.clientY + 'px';` の行の後に、以下を追加してください。

```js
const size = Math.random() * 16 + 16;
heart.style.fontSize = size + 'px';

const colors = ['#e05080', '#ff6b9d', '#ff4444', '#ff8c00', '#ff69b4'];
heart.style.color = colors[Math.floor(Math.random() * colors.length)];

const dx = (Math.random() - 0.5) * 120;
const dy = -(Math.random() * 80 + 60);
heart.style.setProperty('--dx', dx + 'px');
heart.style.setProperty('--dy', dy + 'px');
```

:::note info
ボタンを押すたびにハートがバラバラの方向・サイズ・色で飛び散ることを確認してください。
:::


## 磨き上げよう

#### ポイント①：ハートをバラバラのタイミングで飛ばす

`for` ループ内に遅延を加えると、全員が同時ではなくバラバラに飛び出して自然に見えます。JSを下のコードに書き換えてください。

```js
btn.addEventListener('click', function(event) {
  count = count + 1;
  countDisplay.textContent = count;

  for (let i = 0; i < 8; i++) {
    const delay = i * 50;

    setTimeout(function() {
      const heart = document.createElement('div');
      heart.textContent = '♥';
      heart.classList.add('heart');
      heart.style.left = event.clientX + 'px';
      heart.style.top  = event.clientY + 'px';

      const size = Math.random() * 16 + 16;
      heart.style.fontSize = size + 'px';

      const colors = ['#e05080', '#ff6b9d', '#ff4444', '#ff8c00', '#ff69b4'];
      heart.style.color = colors[Math.floor(Math.random() * colors.length)];

      const dx = (Math.random() - 0.5) * 120;
      const dy = -(Math.random() * 80 + 60);
      heart.style.setProperty('--dx', dx + 'px');
      heart.style.setProperty('--dy', dy + 'px');

      document.body.appendChild(heart);
      heart.classList.add('flying');

      setTimeout(function() {
        heart.remove();
      }, 1000);
    }, delay);
  }
});
```

`delay = i * 50` にすると、1個目は即座、2個目は50ms後、3個目は100ms後…と順番に出てきます。

#### ポイント②：ボタンを押したときにページを揺らす

CSSの一番下に追加してください。

```css
@keyframes shake {
  0%   { transform: translate(0, 0); }
  25%  { transform: translate(-4px, 0); }
  75%  { transform: translate(4px, 0); }
  100% { transform: translate(0, 0); }
}

.shaking {
  animation: shake 0.2s ease-out;
}
```

クリックイベントの中の先頭（`for` ループの前）に、以下を追加してください。

```js
const page = document.querySelector('.page');
page.classList.add('shaking');
page.addEventListener('animationend', function() {
  page.classList.remove('shaking');
}, { once: true });
```
