## 「クリックで召喚！出現の魔法を習得せよ」

## ミッション：画面をクリックすると、クリックした場所に丸が出るようにしよう！

この課題では「要素を自分で作ってページに追加する」「クリックした座標を取得する」という2つの技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">画面のどこかをクリックしてみよう</p>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #f0f4ff;
  min-height: 100vh;
}

.hint {
  text-align: center;
  color: #999;
  padding: 20px;
}

.dot {
  position: fixed;
  width: 20px;
  height: 20px;
  background: #5080e0;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  pointer-events: none;
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  const dot = document.createElement('div');
  dot.classList.add('dot');
  dot.style.left = event.clientX + 'px';
  dot.style.top  = event.clientY + 'px';
  document.body.appendChild(dot);
});
```

画面のどこかをクリックすると、そこに青い丸が出ることを確認してください。

### コードを読み解こう

#### `document.addEventListener('click', ...)` ：画面全体のクリックを検知する

```js
document.addEventListener('click', function(event) {
  // クリックするたびにここが実行される
});
```

`document` は「ページ全体」を表すオブジェクトです。`document` にイベントリスナーを付けると、ページのどこをクリックしても反応します。

#### クリック座標を取得する

```js
function(event) {
```

`function` のカッコに `event` を書くと、クリックの情報（座標など）を受け取れるようになります。

```js
dot.style.left = event.clientX + 'px';
dot.style.top  = event.clientY + 'px';
```

| プロパティ | 意味 |
|-----------|------|
| `event.clientX` | クリックした場所のX座標（左からのピクセル数） |
| `event.clientY` | クリックした場所のY座標（上からのピクセル数） |

#### 要素を自分で作ってページに追加する3ステップ

```js
const dot = document.createElement('div');  // 1. 空のdivを作る
dot.classList.add('dot');                   // 2. CSSのクラスを付ける
document.body.appendChild(dot);            // 3. bodyの中に追加する
```

`createElement` で箱を作り、`classList.add` でスタイルを着せて、`appendChild` でページに出す、という流れです。

#### `position: fixed` + `transform: translate(-50%, -50%)`

```css
.dot {
  position: fixed;
  transform: translate(-50%, -50%);
}
```

`position: fixed` にすると `left` / `top` で座標を指定できます。ただしこのままだと丸の左上角がクリック位置に来てしまいます。`transform: translate(-50%, -50%)` で丸の中心がクリック位置に来るように調整しています。

#### `pointer-events: none`

```css
pointer-events: none;
```

出した丸をクリックしてもイベントが発生しないようにします。これがないと、出した丸をクリックしたときに「丸の上をクリックした」として扱われてしまい、意図しない動作になることがあります。

### 改造しよう

#### ミッション①：丸の色を変えよう

```css
.dot {
  background: #5080e0;  ← 好きな色に変える
}
```

#### ミッション②：丸のサイズを変えよう

```css
.dot {
  width: 20px;   ← 大きくしたければ数字を増やす
  height: 20px;  ← widthと同じ値にすると丸のまま
}
```

#### ミッション③：丸以外の形にしよう

`border-radius` を変えると形が変わります。

| 値 | 形 |
|----|----|
| `50%` | 丸 |
| `0` | 四角 |
| `8px` | 角丸四角 |


#### チェックしてみよう

✅️ 画面をクリックすると丸が出る
✅️ クリックした場所に丸が出る（ずれていない）
✅️ `document.createElement` を使っている
✅️ `event.clientX` / `event.clientY` を使っている
