## 「動け！アニメーション魔法を起動せよ」

## ミッション：ボタンを押すたびに、箱がバウンドするアニメーションを動かそう！

この課題では「`@keyframes` でアニメーションを定義する」「JSからクラスを付けてアニメーションを発動させる」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<div class="stage">
  <div id="box">🎁</div>
  <button id="btn">揺らす</button>
</div>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #fff8f0;
  font-family: sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

.stage {
  text-align: center;
}

#box {
  font-size: 5rem;
  display: inline-block;
  margin-bottom: 24px;
}

#btn {
  display: block;
  margin: 0 auto;
  padding: 12px 32px;
  font-size: 1rem;
  background: #ff8c00;
  color: white;
  border: none;
  border-radius: 50px;
  cursor: pointer;
}

@keyframes bounce {
  0%   { transform: translateY(0); }
  30%  { transform: translateY(-40px); }
  60%  { transform: translateY(-20px); }
  80%  { transform: translateY(-8px); }
  100% { transform: translateY(0); }
}

.bouncing {
  animation: bounce 0.6s ease-out;
}
```

JSタブに下のコードを貼り付けてください。

```js
const box = document.querySelector('#box');
const btn = document.querySelector('#btn');

btn.addEventListener('click', function() {
  box.classList.add('bouncing');

  box.addEventListener('animationend', function() {
    box.classList.remove('bouncing');
  }, { once: true });
});
```

「揺らす」ボタンを押すたびに箱がバウンドすることを確認してください。

### コードを読み解こう

#### `@keyframes` ：アニメーションの動きを定義する

```css
@keyframes bounce {
  0%   { transform: translateY(0); }
  30%  { transform: translateY(-40px); }
  60%  { transform: translateY(-20px); }
  80%  { transform: translateY(-8px); }
  100% { transform: translateY(0); }
}
```

`@keyframes アニメーション名 { }` でアニメーションの動きを定義します。`0%` が開始、`100%` が終了で、途中の数字でどの時点でどの状態にするかを指定します。

`translateY` は上下方向の移動で、マイナスが上です。

#### クラスを付けてアニメーションを発動させる

```css
.bouncing {
  animation: bounce 0.6s ease-out;
}
```

```js
box.classList.add('bouncing');
```

`animation` プロパティはクラスが付いた瞬間に再生が始まります。JSで `.bouncing` クラスを付けることでアニメーションを発動させています。

| `animation` の値 | 意味 |
|-----------------|------|
| `bounce` | `@keyframes` で定義した動きの名前 |
| `0.6s` | アニメーションの長さ |
| `ease-out` | 終わりに向かってゆっくりになる |

#### アニメーション終了後にクラスを外す

```js
box.addEventListener('animationend', function() {
  box.classList.remove('bouncing');
}, { once: true });
```

`animationend` はアニメーションが終わったときに発火するイベントです。終了後にクラスを外しておかないと、次にボタンを押したときに「すでにクラスが付いている」状態になり、アニメーションが再生されません。

`{ once: true }` は「1回だけ実行したら自動でリスナーを外す」オプションです。

### 改造しよう

#### ミッション①：アニメーションの速さを変えよう

```css
.bouncing {
  animation: bounce 0.6s ease-out;
                    ↑ ここを変える
}
```

`0.3s` にするとキビキビ、`1s` にするとゆっくりになります。

#### ミッション②：動きを変えよう

`translateY` の値を変えると飛ぶ高さが変わります。左右に揺らしたいときは `translateX` を使います。

```css
@keyframes shake {
  0%   { transform: translateX(0); }
  25%  { transform: translateX(-10px); }
  75%  { transform: translateX(10px); }
  100% { transform: translateX(0); }
}
```

#### ミッション③：絵文字を変えよう

```html
<div id="box">🎁</div>
                ↑ 好きな絵文字に変える
```


#### チェックしてみよう

✅️ ボタンを押すと箱がバウンドする
✅️ 何度押しても毎回アニメーションが再生される
✅️ `@keyframes` でアニメーションを定義している
✅️ `classList.add` でアニメーションを発動させている
✅️ `animationend` でクラスを外している
