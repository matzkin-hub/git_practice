## 「自動消滅！消えるタイマー魔法を習得せよ」

## ミッション：クリックすると星が出て、1秒後に自動で消えるようにしよう！

この課題では「`setTimeout` で一定時間後に処理を実行する」「`remove()` で要素を削除する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">画面をクリックすると星が出て、消えます</p>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #0a0a1a;
  min-height: 100vh;
}

.hint {
  text-align: center;
  color: #555;
  padding: 20px;
}

.star {
  position: fixed;
  font-size: 2rem;
  transform: translate(-50%, -50%);
  pointer-events: none;
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  const star = document.createElement('div');
  star.textContent = '⭐';
  star.classList.add('star');
  star.style.left = event.clientX + 'px';
  star.style.top  = event.clientY + 'px';
  document.body.appendChild(star);

  setTimeout(function() {
    star.remove();
  }, 1000);
});
```

画面をクリックすると星が出て、1秒後に消えることを確認してください。

### コードを読み解こう

#### `setTimeout` ：一定時間後に処理を実行する

```js
setTimeout(function() {
  // ここに遅れて実行したい処理を書く
}, 1000);
```

`setTimeout(処理, 時間)` は「指定した時間（ミリ秒）が経ったあとに処理を1回だけ実行する」命令です。

`1000` ミリ秒 = 1秒です。

| 時間の値 | 実際の秒数 |
|---------|----------|
| `500` | 0.5秒後 |
| `1000` | 1秒後 |
| `2000` | 2秒後 |
| `3000` | 3秒後 |

#### `remove()` ：要素を削除する

```js
star.remove();
```

要素に対して `remove()` を呼ぶと、その要素がページから削除されます。

クリックするたびに星を追加し続けると、削除しなければページに星が増え続けます。`setTimeout` と `remove()` を組み合わせて「一定時間後に自動で消える」動きを作っています。

#### `setTimeout` の中の処理は「後で」実行される

```js
const star = document.createElement('div');
star.textContent = '⭐';
// ... （省略）
document.body.appendChild(star);   // ← まず星を出す

setTimeout(function() {
  star.remove();                   // ← 1秒後に消す
}, 1000);
```

`setTimeout` の中の処理はすぐには実行されません。`appendChild` で星を出してから、1秒待って `remove()` が実行されます。

### 改造しよう

#### ミッション①：消えるまでの時間を変えよう

```js
setTimeout(function() {
  star.remove();
}, 1000);  ← ここを変える
```

`500` にすると0.5秒、`3000` にすると3秒で消えます。

#### ミッション②：絵文字を変えよう

```js
star.textContent = '⭐';  ← 好きな絵文字に変える
```

`'🌸'`、`'💥'`、`'🔥'` など、好きなものを試してみましょう。

#### ミッション③（チャレンジ）：消える直前にフェードアウトさせよう

CSS に追加：

```css
.star {
  transition: opacity 0.3s;
}

.star.fade {
  opacity: 0;
}
```

JS を変更：

```js
setTimeout(function() {
  star.classList.add('fade');
  setTimeout(function() {
    star.remove();
  }, 300);
}, 700);
```

0.7秒後にフェードアウト開始、0.3秒かけて消えて、合計1秒後に削除、という流れです。

✅️ `setTimeout` を使っている
✅️ `remove()` を使っている
