## 「カオス解放！ランダム魔法を操れ」

## ミッション：クリックするたびに、違う色・サイズ・方向で丸が飛び散るようにしよう！

この課題では「`Math.random()` で毎回違う値を作る」「CSS変数をJSから操作する」技術を習得します。

### コードを写そう

HTMLタブに下のコードを貼り付けてください。

```html
<p class="hint">画面をクリックしてみよう</p>
```

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  background: #111;
  min-height: 100vh;
}

.hint {
  text-align: center;
  color: #555;
  padding: 20px;
}

.dot {
  position: fixed;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  pointer-events: none;
}

.dot.flying {
  animation: fly 0.8s ease-out forwards;
}

@keyframes fly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(
      calc(-50% + var(--dx)),
      calc(-50% + var(--dy))
    );
  }
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  for (let i = 0; i < 8; i++) {
    const dot = document.createElement('div');
    dot.classList.add('dot');
    dot.style.left = event.clientX + 'px';
    dot.style.top  = event.clientY + 'px';

    // ランダムなサイズ（10〜30px）
    const size = Math.random() * 20 + 10;
    dot.style.width  = size + 'px';
    dot.style.height = size + 'px';

    // ランダムな色
    const colors = ['#ff4444', '#ff8c00', '#ffdd00', '#44ff88', '#4488ff', '#cc44ff'];
    dot.style.background = colors[Math.floor(Math.random() * colors.length)];

    // ランダムな飛ぶ方向
    const dx = (Math.random() - 0.5) * 150;
    const dy = (Math.random() - 0.5) * 150;
    dot.style.setProperty('--dx', dx + 'px');
    dot.style.setProperty('--dy', dy + 'px');

    document.body.appendChild(dot);
    dot.classList.add('flying');

    setTimeout(function() {
      dot.remove();
    }, 800);
  }
});
```

画面をクリックするたびに、バラバラの色・サイズ・方向で丸が飛び散ることを確認してください。

### コードを読み解こう

#### `Math.random()` ：0以上1未満のランダムな数を返す

```js
Math.random()  // 例：0.7341...、0.0829...、0.9912...
```

実行するたびに毎回違う値が返ってきます。そのままでは小さすぎるので、欲しい範囲に合わせて計算します。

**範囲を変える計算式：**

```js
// 0〜20の範囲
Math.random() * 20         // 例：14.3、2.7、19.1

// 10〜30の範囲（最小値を足す）
Math.random() * 20 + 10    // 例：24.3、12.7、29.1

// -75〜+75の範囲（マイナスを含む）
(Math.random() - 0.5) * 150  // 例：+43.1、-61.2、+12.8
```

#### 配列からランダムに1つ選ぶ

```js
const colors = ['#ff4444', '#ff8c00', '#ffdd00', '#44ff88', '#4488ff', '#cc44ff'];
dot.style.background = colors[Math.floor(Math.random() * colors.length)];
```

`colors.length` は配列の要素数（ここでは6）です。`Math.random() * 6` で0〜6未満のランダムな数を作り、`Math.floor` で小数点以下を切り捨てて整数（0〜5）にします。これで配列のインデックスとして使えます。

| 式 | 結果例 |
|----|--------|
| `Math.random() * 6` | `3.71...` |
| `Math.floor(3.71)` | `3` |
| `colors[3]` | `'#44ff88'`（緑） |

#### CSS変数（`--dx`, `--dy`）：JSからCSSに値を渡す

```css
@keyframes fly {
  to {
    transform: translate(
      calc(-50% + var(--dx)),
      calc(-50% + var(--dy))
    );
  }
}
```

```js
dot.style.setProperty('--dx', dx + 'px');
dot.style.setProperty('--dy', dy + 'px');
```

`var(--dx)` は「`--dx` という名前のスロットに入っている値を使う」という意味です。JSから `setProperty('--dx', ...)` でスロットに値を入れることで、要素ごとに違う方向に飛ばせます。

```
丸1: --dx = -45px, --dy = +30px  → 左下に飛ぶ
丸2: --dx = +70px, --dy = -60px  → 右上に飛ぶ
丸3: --dx = -10px, --dy = +75px  → ほぼ真下に飛ぶ
```
