## 「精霊を呼び出す場所」

**ミッション：画面のクリックした場所に丸が現れる装置をつくろう！**

### コードを写そう

HTMLタブは空のままで構いません。

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  min-height: 100vh;
  background: #111;
}

.spirit {
  position: fixed;
  width: 40px;
  height: 40px;
  background: gold;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  pointer-events: none;
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  const spirit = document.createElement('div');
  spirit.classList.add('spirit');
  spirit.style.left = event.clientX + 'px';
  spirit.style.top = event.clientY + 'px';
  document.body.appendChild(spirit);
});
```

画面のどこかをクリックして、クリックした場所に金色の丸が現れることを確認してください。

### コードを読み解こう

#### `document.addEventListener('click', function(event) { ... })`：ページ全体のクリックを検知する

```js
document.addEventListener('click', function(event) {
  // ここがクリックされるたびに実行される
});
```

ここでは `document`（ページ全体）にクリックイベントを付けています。ページのどこをクリックしても反応します。

#### `event` パラメータ：クリックの「情報パッケージ」

```js
function(event) { ... }
```

`event` は「クリックしたときの情報が詰まったパッケージ」です。関数の引数として受け取ります。

このパッケージの中には、クリックした場所の座標が入っています。

| プロパティ | 意味 |
|-----------|------|
| `event.clientX` | クリックした場所の横座標（画面左端から何ピクセルか） |
| `event.clientY` | クリックした場所の縦座標（画面上端から何ピクセルか） |

#### `spirit.style.left` / `spirit.style.top`：要素の位置を直接指定する

```js
spirit.style.left = event.clientX + 'px';
spirit.style.top  = event.clientY + 'px';
```

`style.left` は要素の左端の位置、`style.top` は要素の上端の位置を指定します。`+ 'px'` で単位をつけています。

**なぜ `'px'` を文字列でつなげるの？**

```js
// 例：event.clientX が 300 のとき
spirit.style.left = 300 + 'px';  // → '300px' という文字列になる
```

CSSでは `left: 300px;` のように単位が必要なため、数字に `'px'` をつなげています。

#### `position: fixed`：画面上の特定の場所に固定して配置する

```css
.spirit {
  position: fixed;
}
```

`position: fixed` を使うと、`left` と `top` で「画面の左端・上端からの距離」を指定して要素を配置できます。`event.clientX/Y` も「画面の左端・上端からの距離」なので、そのまま使えます。

#### `transform: translate(-50%, -50%)`：要素の中心をクリック位置に合わせる

```css
transform: translate(-50%, -50%);
```

`left` と `top` は要素の**左上の角**の位置を指定します。このままだと丸の左上がクリック位置に来て、ズレて見えます。

`translate(-50%, -50%)` は「要素を自分のサイズの半分だけ左上に動かす」という命令です。これで要素の**中心**がクリック位置に来ます。

### 改造しよう

#### ミッション①：丸の色を変えよう

```css
.spirit {
  background: gold;  ← ここを変える
}
```


#### ミッション②：丸のサイズを変えよう

```css
.spirit {
  width: 40px;   ← ここを変える
  height: 40px;  ← ここも変える（widthと同じ値にすると丸になる）
}
```
