## 「ランダム魔法陣」

**ミッション：クリックするたびに、毎回違うサイズの丸が現れる装置をつくろう！**

### コードを写そう

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  min-height: 100vh;
  background: #111;
}

.spirit {
  position: fixed;
  background: gold;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  pointer-events: none;
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  const spirit = document.createElement('div');
  spirit.classList.add('spirit');
  spirit.style.left = event.clientX + 'px';
  spirit.style.top  = event.clientY + 'px';

  const size = Math.random() * 60 + 20;
  spirit.style.width  = size + 'px';
  spirit.style.height = size + 'px';

  document.body.appendChild(spirit);
});
```

クリックするたびに異なるサイズの丸が現れることを確認してください。

### コードを読み解こう

`width` と `height` を決まった値ではなく毎回ランダムな値にしています。

#### `Math.random()`：ランダムな小数を生み出す

```js
Math.random()
```

`Math.random()` を呼ぶたびに、`0` 以上 `1` 未満のランダムな小数が返ってきます。

```
Math.random() の結果の例：
  0.3271...
  0.8149...
  0.0523...
  0.9987...
```

毎回違う値なので、これを使うと「毎回違う結果」を作れます。

#### `Math.random() * 60 + 20`：範囲を決める

```js
const size = Math.random() * 60 + 20;
```

`Math.random()` の結果は 0〜0.999... なので、そのままでは小さすぎます。範囲を調整する公式があります。

```
Math.random() * 幅 + 最小値
```

| 式 | 結果の範囲 |
|----|----------|
| `Math.random() * 60` | 0〜60 |
| `Math.random() * 60 + 20` | 20〜80 |

`+ 20` で最小サイズ20pxを保証しています。最小値を決めることで、極端に小さい丸が出ないようにしています。

#### `spirit.style.width = size + 'px'`：計算した値をスタイルに使う

```js
spirit.style.width  = size + 'px';
spirit.style.height = size + 'px';
```

ここでは毎回計算した値をJSから直接指定しています。`width` と `height` を同じ値にすると正円になります。

### 改造しよう

#### ミッション①：サイズの範囲を変えよう

```js
const size = Math.random() * 60 + 20;
                             ↑     ↑
                            幅   最小値
```

`60` と `20` を変えて、サイズの範囲を好きな値にしてみましょう。


#### ミッション②：色もランダムにしよう

`colors` という配列（リスト）を使って、ランダムに色を選べます。

JSに下のコードを追加してみましょう（`document.body.appendChild(spirit);` の前に追加する）。

```js
const colors = ['gold', 'tomato', 'deepskyblue', 'limegreen', 'violet'];
const randomIndex = Math.floor(Math.random() * colors.length);
spirit.style.background = colors[randomIndex];
```

**コードの読み方：**

- `colors` は色の名前を並べたリスト（配列）
- `colors.length` はリストの個数（この場合 `5`）
- `Math.random() * 5` で 0〜4.999... のランダムな数を作る
- `Math.floor(...)` で小数点以下を切り捨て、0〜4 の整数にする
- `colors[0]` が `'gold'`、`colors[1]` が `'tomato'`、... のように番号でリストから取り出せる
