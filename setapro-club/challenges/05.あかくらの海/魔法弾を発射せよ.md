## 「魔法弾を発射せよ」

**ミッション：画面をクリックすると、その場所から魔法弾（丸）が飛び上がって消えるようにしよう！**

### 完成条件

- 画面のどこかをクリックすると、その位置に円が出現する
- 円はアニメーションで上に飛んでいき、フェードアウトして消える
- クリックするたびに新しい円が生まれる
- 円のサイズや色が毎回ランダムに変わる


### これまでに学んだ技術を組み合わせよう

この中ボスは これまでの技術をすべて組み合わせます。

| 使う技術 | どこで学んだか |
|----------|-------------|
| `@keyframes` + `animation` | 「変身の呪い」 |
| `classList.add` でアニメーション発動 | 「呪いを外から掛ける」 |
| `setTimeout` + `remove()` で消える | 「消えていく幻影」 |
| `event.clientX/Y` + `position: fixed` | 「精霊を呼び出す場所」 |
| `Math.random()` でランダム | 「ランダム魔法陣」 |


### コードを写そう

CSSタブに下のコードを貼り付けてください。

```css
body {
  margin: 0;
  min-height: 100vh;
  background: #111;
}

.bullet {
  position: fixed;
  border-radius: 50%;
  transform: translate(-50%, -50%);
  pointer-events: none;
}

.flying {
  animation: fly 1s ease-out forwards;
}

@keyframes fly {
  from {
    opacity: 1;
    transform: translate(-50%, -50%);
  }
  to {
    opacity: 0;
    transform: translate(-50%, calc(-50% - 150px));
  }
}
```

JSタブに下のコードを貼り付けてください。

```js
document.addEventListener('click', function(event) {
  // 1. 魔法弾を工作する
  const bullet = document.createElement('div');
  bullet.classList.add('bullet');

  // 2. クリックした場所に置く
  bullet.style.left = event.clientX + 'px';
  bullet.style.top  = event.clientY + 'px';

  // 3. ランダムなサイズを決める
  const size = Math.random() * 40 + 20;
  bullet.style.width  = size + 'px';
  bullet.style.height = size + 'px';

  // 4. ランダムな色を決める
  const colors = ['gold', 'tomato', 'deepskyblue', 'limegreen', 'violet'];
  bullet.style.background = colors[Math.floor(Math.random() * colors.length)];

  // 5. 舞台に出す
  document.body.appendChild(bullet);

  // 6. アニメーション開始
  bullet.classList.add('flying');

  // 7. 1秒後に消す
  setTimeout(function() {
    bullet.remove();
  }, 1000);
});
```

画面をクリックして魔法弾が飛び上がって消えることを確認してください。

### コードを読み解こう

#### `animation: fly 1s ease-out forwards`：アニメーションの新しいキーワード

今回は1回だけ再生して止まります。

| キーワード | 意味 |
|-----------|------|
| `1s` | 1秒かけてアニメーション |
| `ease-out` | 最初は速く、最後はゆっくり終わる（自然な動き） |
| `forwards` | アニメーション終了後、最後の状態（`to`）のまま止まる |

`forwards` がないと、アニメーション後に元の位置・透明度に戻ってしまいます。

#### `calc(-50% - 150px)`：計算式をCSSに書く

```css
to {
  transform: translate(-50%, calc(-50% - 150px));
}
```

`calc()` はCSSの中で計算ができる機能です。`-50%` は「要素の中心に合わせる」、`- 150px` は「さらに150px上に移動する」という意味です。合わせて「中心に合わせた位置から150px上」になります。

### 改造しよう

#### ミッション①：飛ぶ高さを変えよう

```css
transform: translate(-50%, calc(-50% - 150px));
                                          ↑ ここを変える
```

`150px` を大きくするともっと高く飛びます。


#### ミッション②：アニメーションのスピードを変えよう

```css
.flying {
  animation: fly 1s ease-out forwards;
                  ↑ ここを変える
}
```

`1s` を変えて速さを調整してみましょう。`setTimeout` の `1000` も合わせて変えると消えるタイミングがアニメーションに合います。


#### チャレンジミッション：中ボス完成宣言

自分だけのカスタマイズを1つ以上加えて、「完成」を宣言しよう。

例：
- 好きな絵文字を丸の中に入れる
- 背景色を変える
- 丸の数を増やす（`for` ループが使えます）
